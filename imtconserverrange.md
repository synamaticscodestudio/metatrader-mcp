IMTConServerRange
The IMTConServerRange interface contains methods for setting the ranges of accounts, orders and deals on trade servers.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconserverrange_release.md "Release") | Delete the current object. |
| [Assign](imtconserverrange_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconserverrange_clear.md "Clear") | Clear an object. |
| [From](imtconserverrange_from.md "From") | Get and set the beginning of the range of accounts, orders or deals of a trade server. |
| [To](imtconserverrange_to.md "To") | Get and set the end of the range of accounts, orders or deals of a trade server. |
| [UsedFrom](imtconserverrange_usedfrom.md "UsedFrom") | Gets the beginning of the range of accounts, orders or deals already used on a trade server. |
| [UsedTo](imtconserverrange_usedto.md "UsedTo") | Get the end of the range of accounts, orders or deals already used on a trade server. |

Trade Requests
Interfaces provide access to managing trade requests received on the server, as well as allow creating requests.
The following interfaces are available for working with trade requests:
  * [IMTRequest](imtrequest.md "IMTRequest") An interface that provides access to the parameters of trade requests.
  * [IMTRequestArray](imtrequestarray.md "IMTRequestArray") An interface for working with the arrays of trade requests.
  * [IMTRequestSink](imtrequestsink.md "IMTRequestSink") An interface that contains handlers of events associated with trade requests.
  * [IMTConfirm](imtconfirm.md "IMTConfirm") An interface for confirming trade requests.
  * [IMTExecution](imtexecution.md "IMTExecution") A trade execution interface.

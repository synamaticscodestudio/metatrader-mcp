Depth of Market
The MetaTrader 5 Server API provides means for working with the Depth of Market. Using the functions you can receive information from the Depth of Market, as well as subscribe and unsubscribe from events associated with its change.
Depth of market functions and interfaces work only on history servers.  
---  
| Function | Purpose |
| --- | --- |
| --- | --- |
| [BookSubscribe](imtserverapi_booksubscribe.md "BookSubscribe") | Subscribe to events associated with changes in the symbol's DOM. |
| [BookUnsubscribe](imtserverapi_bookunsubscribe.md "BookUnsubscribe") | Unsubscribe from events associated with changes in the symbol's DOM. |
| [BookGet](imtserverapi_bookget.md "BookGet") | Get a symbol's Depth of Market. |

Configuration of Symbols
To get the symbols configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [SymbolCreate](imtreportapi_symbolcreate.md "SymbolCreate") | Create an object of the symbol configuration. |
| [SymbolSessionCreate](imtreportapi_symbolsessioncreate.md "SymbolSessionCreate") | Create an object of configuration of a trading or quoting session of the symbol. |
| [SymbolTotal](imtreportapi_symboltotal.md "SymbolTotal") | The total number of symbol configurations available in the platform. |
| [SymbolNext](imtreportapi_symbolnext.md "SymbolNext") | Get the symbol configuration by the index. |
| [SymbolGet](imtreportapi_symbolget.md "SymbolGet") | Get the symbol configuration. |
| [SymbolGetLight](imtreportapi_symbolgetlight.md "SymbolGetLight") | Get an eased configuration of a symbol. |
| [SymbolExist](imtreportapi_symbolexist.md "SymbolExist") | Check the availability of a symbol for a specified group of clients. |

IMTConMessengerGroup
The IMTConMessengerGroup class contains methods for getting and setting group settings in the messenger:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconmessengergroup_release.md "Release") | Delete the current object. |
| [Assign](imtconmessengergroup_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconmessengergroup_clear.md "Clear") | Clear an object. |
| [Group](imtconmessengergroup_group.md "Group") | Get and set the group of accounts for which the messenger is used. |

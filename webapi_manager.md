Managers
The following requests are provided for operations with manager settings:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/manager/add](webapi_manager_add.md "Add") | Add and update a manager configuration in the trading platform. |
| [/api/manager/delete](webapi_manager_delete.md "Delete") | Delete manager configurations from the trading platform. |
| [/api/manager/shift](webapi_manager_shift.md "Shift") | Change the position of a manager configuration in the list. |
| [/api/manager/total](webapi_manager_total.md "Get Total") | Get the total number of manager configurations existing in the platform. |
| [/api/manager/next](webapi_manager_next.md "Get by Index") | Get one or more manager configurations by index in the list. |
| [/api/manager/get](webapi_manager_get.md "Get by Login") | Get a manager configuration by login. |
| [/api/manager/get_all](webapi_manager_get_all.md "Get All") | Get all existing manager configurations. |

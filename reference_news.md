News Database
The MetaTrader 5 Manager API provides interfaces for working with the news database of the platform. Using them you can receive and send news, as well as handle events associated with changes in the news database of the server.
The following interfaces for working with the news database are available:
  * [IMTNews](imtnews.md "IMTNews") An interface for working with news database.
  * [IMTNewsSink](imtnewssink.md "IMTNewsSink") An interface for working with the events of the news database

Constants
The MetaTrader 5 API providers a number of volume and price constants:
| ID | Value | Description |
| --- | --- | --- |
| --- | --- | --- |
| MTAPI_VOLUME_DIV | 10000.0 | A factor for converting the volume. |
| MTAPI_VOLUME_DIGITS | 4 | Precision of the volume value. |
| MTAPI_VOLUME_MAX | (UINT64)10000000000 | The maximum possible value of the volume. |
| MTAPI_PRICE_MAX | 100000000 | The maximum possible value of the price. |
| MTAPI_PRICE_DIGITS_MAX | 8 | The maximum number of digits after the decimal point in the price. |

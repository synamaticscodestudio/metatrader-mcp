IMTConAccountAgreement
The IMTConAccountAgreement class contains methods for working with the [agreements](https://support.metaquotes.net/ru/docs/mt5/platform/administration/admin_accounts/account_allocation_groups#agreement) which are shown to clients when opening accounts through terminals.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconaccountagreement_release.md "Release") | Delete the current object. |
| [Assign](imtconaccountagreement_assign.md "Assign") | Assign the passed object to the current one. |
| [Clear](imtconaccountagreement_clear.md "Clear") | Clear an object. |
| [CaptionType](imtconaccountagreement_captiontype.md "CaptionType") | Get the agreement type. |
| [CaptionCustomText](imtconaccountagreement_captioncustomtext.md "CaptionCustomText") | Get the name of the user agreement. |
| [URL](imtconaccountagreement_url.md "URL") | Get the agreement link. |
| [Flags](imtconaccountagreement_flags.md "Flags") | Get additional agreement settings. |

The IMTConAccountAgreement class contains the following enumerations:
| Enumeration | Purpose |
| --- | --- |
| --- | --- |
| [EnCaptionType](imtconaccountagreement_enum.htm#encaptiontype) | Agreement types. |
| [EnFlags](imtconaccountagreement_enum.htm#enflags) | Additional agreement settings. |

IMTConPluginModule
The IMTConPluginModule class contains methods for getting and changing parameters of plugin modules:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconpluginmodule_release.md "Release") | Delete the current object. |
| [Assign](imtconpluginmodule_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconpluginmodule_clear.md "Clear") | Clear an object. |
| [Name](imtconpluginmodule_name.md "Name") | Get the plugin name, which is inserted by default to a configuration when selecting this module. |
| [Vendor](imtconpluginmodule_vendor.md "Vendor") | Get the name of the plugin module provider. |
| [Description](imtconpluginmodule_description.md "Description") | Get the description of a plugin module. |
| [Module](imtconpluginmodule_module.md "Module") | Get the name of the plugin module file. |
| [Server](imtconpluginmodule_server.md "Server") | Get the ID of the server. |
| [Version](imtconpluginmodule_version.md "Version") | Get the version of a plugin module. |
| [VersionAPI](imtconpluginmodule_versionapi.md "VersionAPI") | Get the version of the Server API. |
| [ParameterTotal](imtconpluginmodule_parametertotal.md "ParameterTotal") | It returns the number of parameters of a plugin module. |
| [ParameterNext](imtconpluginmodule_parameternext.md "ParameterNext") | Get the plugin module parameters by the index. |
| [ParameterGet](imtconpluginmodule_parameterget.md "ParameterGet") | Get the plugin module parameter by the name. |

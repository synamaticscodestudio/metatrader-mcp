Columns Functions
The following functions for managing columns are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [TableColumnCreate](imtreportapi_tablecolumncreate.md "TableColumnCreate") | A column object creation. |
| [TableColumnClear](imtreportapi_tablecolumnclear.md "TableColumnClear") | Clear table columns description. |
| [TableColumnAdd](imtreportapi_tablecolumnadd.md "TableColumnAdd") | Add a column description to a table end. |
| [TableColumnDelete](imtreportapi_tablecolumndelete.md "TableColumnDelete") | Delete a column description from a table by index. |
| [TableColumnTotal](imtreportapi_tablecolumntotal.md "TableColumnTotal") | Get the number of columns in a table. |
| [TableColumnSize](imtreportapi_tablecolumnsize.md "TableColumnSize") | Get a total size of one table row in bytes. |
| [TableColumnNext](imtreportapi_tablecolumnnext.md "TableColumnNext") | Get a column description by index. |

Protocol Implementation in .NET
The MetaTrader 5 SDK package includes an example of [.NET implementation of the Web API protocol](net_mtwebapi.md "MT5WebAPI Class"). The implementation is available in the form of a ready to use DLLs MetaQuotes.MT5WebAPI.dll. It is are available in the [MetaTrader 5 SDK installation directory]\Libs\\. 
Include the library in your project to use the functions. It is suitable for both 32 and 64 bit applications. The full description of these function is provided under the [MT5WebAPI Class](net_mtwebapi.md "MT5WebAPI Class") section.
In addition, the package includes [.NET source code for a ready to use website](net_webtrader.md "WebTrader") demonstrating the Web API capabilities. Example is available as a Microsoft Visual Studio project. It is available in the [MetaTrader 5 SDK installation directory]\Examples\Web\WebTrader\\.

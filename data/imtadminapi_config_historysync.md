Configuration of History Synchronization
Functions allow managing the configurations of history data synchronization, as well subscribe and unsubscribe from events associated with its change.
To configure the synchronization of price data, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [HistorySyncCreate](imtadminapi_historysynccreate.md "HistorySyncCreate") | Create an object of configuration of price data synchronization. |
| [HistorySyncSubscribe](imtadminapi_historysyncsubscribe.md "HistorySyncSubscribe") | Subscribe to events associated with the configuration of price data synchronization. |
| [HistorySyncUnsubscribe](imtadminapi_historysyncunsubscribe.md "HistorySyncUnsubscribe") | Unsubscribe from events associated with the configuration of price data synchronization. |
| [HistorySyncStart](imtadminapi_historysyncstart.md "HistorySyncStart") | Start synchronization of historical data. |
| [HistorySyncUpdate](imtadminapi_historysyncupdate.md "HistorySyncUpdate") | Add or update of a configuration of price data synchronization. |
| [HistorySyncUpdateBatch](imtadminapi_historysyncupdatebatch.md "HistorySyncUpdateBatch") | Add and edit multiple configurations of price data synchronization. |
| [HistorySyncDelete](imtadminapi_historysyncdelete.md "HistorySyncDelete") | Delete a configuration of price data synchronization by its index. |
| [HistorySyncDeleteBatch](imtadminapi_historysyncdeletebatch.md "HistorySyncDeleteBatch") | Delete multiple configurations of price data synchronization. |
| [HistorySyncShift](imtadminapi_historysyncshift.md "HistorySyncShift") | Change the position of the configuration of price data synchronization in the list. |
| [HistorySyncTotal](imtadminapi_historysynctotal.md "HistorySyncTotal") | The total number of configurations of price data synchronization available in the platform. |
| [HistorySyncNext](imtadminapi_historysyncnext.md "HistorySyncNext") | Get a configuration of price data synchronization based on its index. |

Automations
The Automations service enables the execution of various actions in the platform based on predefined scenarios. To automate routine operations, select conditions and appropriate actions that should trigger under these conditions. Automations allow companies to streamline numerous day-to-day operations and eventually reduce manual work. For more details, please read the [MetaTrader 5 Administrator Help](https://support.metaquotes.net/en/docs/mt5/platform/administration/automation).
The functions described in this section enable users to manage automation configurations, as well as to subscribe and to unsubscribe from events related to configuration changes.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [AutomationCreate](imtadminapi_automationcreate.md "AutomationCreate") | Create an automation configuration object. |
| [AutomationConditionCreate](imtadminapi_automationconditioncreate.md "AutomationConditionCreate") | Create an automation task condition object. |
| [AutomationActionCreate](imtadminapi_automationactioncreate.md "AutomationActionCreate") | Create an automation task action object. |
| [AutomationParamCreate](imtadminapi_automationparamcreate.md "AutomationParamCreate") | Create an automation condition parameter object. |
| [AutomationSubscribe](imtadminapi_automationsubscribe.md "AutomationSubscribe") | Subscribe to events and hooks associated with automation configurations. |
| [AutomationUnsubscribe](imtadminapi_automationunsubscribe.md "AutomationUnsubscribe") | Unsubscribe from events and hooks associated with the automation configuration. |
| [AutomationUpdate](imtadminapi_automationupdate.md "AutomationUpdate") | Add or update an automation configuration. |
| [AutomationUpdateBatch](imtadminapi_automationupdatebatch.md "AutomationUpdateBatch") | Add or edit multiple automation configurations. |
| [AutomationDelete](imtadminapi_automationdelete.md "AutomationDelete") | Delete an automation configuration by the name or index |
| [AutomationDeleteBatch](imtadminapi_automationdeletebatch.md "AutomationDeleteBatch") | Delete multiple automation configurations. |
| [AutomationShift](imtadminapi_automationshift.md "AutomationShift") | Change the position of an automation configuration in the list. |
| [AutomationTotal](imtadminapi_automationtotal.md "AutomationTotal") | Get the total number of automation configurations available in the platform. |
| [AutomationNext](imtadminapi_automationnext.md "AutomationNext") | Get an automation configuration by index. |
| [AutomationGet](imtadminapi_automationget.md "AutomationGet") | Get an automation configuration by name. |

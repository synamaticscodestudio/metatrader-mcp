Trade Request Routing
The following requests are provided for operations with routing settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/route/add](webapi_route_add.md "Add") | Add and update a routing rule in the trading platform. |
| [/api/route/delete](webapi_route_delete.md "Delete") | Delete routing rules from the trading platform. |
| [/api/route/shift](webapi_route_shift.md "Shift") | Changes the position of a routing rule in the list. |
| [/api/route/total](webapi_route_total.md "Get Total") | Get the total number of routing rules existing in the platform. |
| [/api/route/next](webapi_route_next.md "Get by Index") | Get one or more routing rules by index in the list. |
| [/api/route/get](webapi_route_get.md "Get by Name") | Get a routing rule by its name. |
| [/api/route/get_all](webapi_route_get_all.md "Get All") | Get all existing routing rules. |

Totals Functions
To manage a table totals row, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [TableSummaryCreate](imtreportapi_tablesummarycreate.md "TableSummaryCreate") | Create an object of a totals row cell. |
| [TableSummaryClear](imtreportapi_tablesummaryclear.md "TableSummaryClear") | Clear all total records. |
| [TableSummaryAdd](imtreportapi_tablesummaryadd.md "TableSummaryAdd") | Add a totals row cell into a table. |
| [TableSummaryDelete](imtreportapi_tablesummarydelete.md "TableSummaryDelete") | Delete a cell in a table totals row by its index. |
| [TableSummaryNext](imtreportapi_tablesummarynext.md "TableSummaryNext") | Get the cells of a table totals row by its index. |
| [TableSummaryTotal](imtreportapi_tablesummarytotal.md "TableSummaryTotal") | Get the total amount of totals cells in a table. |

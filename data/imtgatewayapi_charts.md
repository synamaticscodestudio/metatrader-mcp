History Data
The MetaTrader 5 Gateway API provides functions for working with historical price data of the platform that are available in the form of minute bars. They allow you to edit or delete minute bars.
Functions for working with historical data:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ChartRequest](imtgatewayapi_chartrequest.md "ChartRequest") | Request minute bars for a symbol. |
| [ChartDelete](imtgatewayapi_chartdelete.md "ChartDelete") | Delete a bar by the symbol. |
| [ChartUpdate](imtgatewayapi_chartupdate.md "ChartUpdate") | Change historical data of a symbol. |
| [ChartReplace](imtgatewayapi_chartreplace.md "ChartReplace") | Full replacement of history data in the specified period with the passed data. |

ECN
MetaTrader 5 API provides access to [built-in ECN](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn) databases. Functions from these section enable the detailed analysis of the current matching queue and history, of operations with liquidity providers and other data.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ECNCreateMatching](imtadminapi_ecncreatematching.md "ECNCreateMatching") | Create a matching order object. |
| [ECNCreateMatchingArray](imtadminapi_ecncreatematchingarray.md "ECNCreateMatchingArray") | Create an array of matching order objects. |
| [ECNCreateFilling](imtadminapi_ecncreatefilling.md "ECNCreateFilling") | Create a filling order object. |
| [ECNCreateFillingArray](imtadminapi_ecncreatefillingarray.md "ECNCreateFillingArray") | Create an array of filling objects. |
| [ECNCreateProvider](imtadminapi_ecncreateprovider.md "ECNCreateProvider") | Create an object of a provider via which orders are executed. |
| [ECNCreateProviderArray](imtadminapi_ecncreateproviderarray.md "ECNCreateProviderArray") | Create an object of the array of providers via which orders are executed. |
| [ECNCreateHistoryMatching](imtadminapi_ecncreatehistorymatching.md "ECNCreateHistoryMatching") | Create an object of a matching order from history. |
| [ECNCreateHistoryMatchingArray](imtadminapi_ecncreatehistorymatchingarray.md "ECNCreateHistoryMatchingArray") | Create an array of matching order objects from history. |
| [ECNCreateHistoryFilling](imtadminapi_ecncreatehistoryfilling.md "ECNCreateHistoryFilling") | Create an object of a filling order from history. |
| [ECNCreateHistoryFillingArray](imtadminapi_ecncreatehistoryfillingarray.md "ECNCreateHistoryFillingArray") | Create an array of filling order objects from history. |
| [ECNCreateHistoryDeal](imtadminapi_ecncreatehistorydeal.md "ECNCreateHistoryDeal") | Create an object of a deal from the ECN history. |
| [ECNCreateHistoryDealArray](imtadminapi_ecncreatehistorydealarray.md "ECNCreateHistoryDealArray") | Create an object of an array of deals from the ECN history. |
| [ECNRequestByGroup](imtadminapi_ecnrequestbygroup.md "ECNRequestByGroup") | Request the current state of matching for the specified client groups. |
| [ECNRequestByLogins](imtadminapi_ecnrequestbylogins.md "ECNRequestByLogins") | Request the current state of matching for the specified logins. |
| [ECNRequestByTickets](imtadminapi_ecnrequestbytickets.md "ECNRequestByTickets") | Request the current state of matching for the specified order tickets. |
| [ECNRequestHistoryByGroup](imtadminapi_ecnrequesthistorybygroup.md "ECNRequestHistoryByGroup") | Request the matching history for the specified client groups. |
| [ECNRequestHistoryByLogins](imtadminapi_ecnrequesthistorybylogins.md "ECNRequestHistoryByLogins") | Request the matching history for the specified logins. |
| [ECNRequestHistoryByTickets](imtadminapi_ecnrequesthistorybytickets.md "ECNRequestHistoryByTickets") | Request the matching history for the specified order tickets. |

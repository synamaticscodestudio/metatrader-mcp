Geo-services
The [IMTGeo](imtgeo.md "IMTGeo") interface provides access to the GeoIP database information on the trading server. It enables access to the information about visitors' country, city, coordinates and internet service providers, based on their IP addresses. The database is automatically updated on the server once a week, keeping the relevant information up to date.
By using these features, you can generate user activity reports and create maps, among others.

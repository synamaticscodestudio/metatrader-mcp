IMTConKYCSink
The IMTConKYCSink class contains event handlers for KYC provider configuration changes:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnKYCAdd](imtconkycsink_onkycadd.md "OnKYCAdd") | The event handler for adding a new KYC provider configuration. |
| [OnKYCUpdate](imtconkycsink_onkycupdate.md "OnKYCUpdate") | The event handler for updating a KYC provider configuration. |
| [OnKYCDelete](imtconkycsink_onkycdelete.md "OnKYCDelete") | The event handler for deleting a KYC provider configuration. |
| [OnKYCSync](imtconkycsink_onkycsync.md "OnKYCSync") | The event handler for synchronizing KYC provider configurations. |

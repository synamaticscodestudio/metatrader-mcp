Symbols
Use the methods described in this section to obtain settings of symbols on a trade server:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [SymbolAdd](net_symboladd.md "SymbolAdd") | Add or change a symbol configuration on the server. |
| [SymbolDelete](net_symboldelete.md "SymbolDelete") | Delete a symbol configuration with the specified name. |
| [SymbolTotal](net_symboltotal.md "SymbolTotal") | Get the number of symbols created on the trade server. |
| [SymbolNext](net_symbolnext.md "SymbolNext") | Get the configuration of a symbol by its index in the list of symbols of the platform. |
| [SymbolGet](net_symbolget.md "SymbolGet") | Gets the symbol configuration by the name. |
| [SymbolGetGroup](net_symbolgetgroup.md "SymbolGetGroup") | Get an individual configuration of a symbol for a group by the name of the symbol. |

Configuration of Groups
Use these functions to obtain user group settings, as well to subscribe and unsubscribe from events associated with their change.
The following functions are available for working with group settings:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [GroupCreate](imtmanagerapi_groupcreate.md "GroupCreate") | Create an object of the group configuration. |
| [GroupCreateArray](imtmanagerapi_groupcreatearray.md "GroupCreateArray") | Create a groups array object. |
| [GroupSymbolCreate](imtmanagerapi_groupsymbolcreate.md "GroupSymbolCreate") | Create an object of symbol configuration for a group. |
| [GroupCommissionCreate](imtmanagerapi_groupcommissioncreate.md "GroupCommissionCreate") | Create an object of commission configuration for a group. |
| [GroupTierCreate](imtmanagerapi_grouptiercreate.md "GroupTierCreate") | Create an object of commission range configuration for a group. |
| [GroupSubscribe](imtmanagerapi_groupsubscribe.md "GroupSubscribe") | Subscribe to events associated with the configuration of groups. |
| [GroupUnsubscribe](imtmanagerapi_groupunsubscribe.md "GroupUnsubscribe") | Unsubscribe from events associated with the configuration of groups. |
| [GroupUpdate](imtmanagerapi_groupupdate.md "GroupUpdate") | Update group configuration. |
| [GroupUpdateBatch](imtmanagerapi_groupupdatebatch.md "GroupUpdateBatch") | Update multiple group configurations. |
| [GroupTotal](imtmanagerapi_grouptotal.md "GroupTotal") | The total number of group configurations available in the platform. |
| [GroupNext](imtmanagerapi_groupnext.md "GroupNext") | Get the group configuration by the index. |
| [GroupGet](imtmanagerapi_groupget.md "GroupGet") | Get the group configuration by the name. |
| [GroupRequest](imtmanagerapi_grouprequest.md "GroupRequest") | Request a group configuration from a server by the name. |
| [GroupRequestArray](imtmanagerapi_grouprequestarray.md "GroupRequestArray") | Request from the server an array of groups by mask. |

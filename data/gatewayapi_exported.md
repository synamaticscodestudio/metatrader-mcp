Exported Functions
MT5APIGatewayAPI.dll library (and its 64-bit version), which is used for the interaction between the application and MetaTrader 5 trading platform, exports several functions:
| Function | Description |
| --- | --- |
| --- | --- |
| [MTGatewayVersion](mtgatewayversion.md "MTGatewayVersion") | Returns Gateway API library version. |
| [MTGatewayCreate](mtgatewaycreate.md "MTGatewayCreate") | Creates a new [IMTGatewayAPI](imtgatewayapi.md "Main Interface") interface object and returns a pointer to it. |
| [MTGatewayCreateLocal](mtgatewaycreatelocal.md "MTGatewayCreateLocal") | Creates a new [IMTGatewayAPI](imtgatewayapi.md "Main Interface") interface object with predetermined parameters and returns a pointer to it. |

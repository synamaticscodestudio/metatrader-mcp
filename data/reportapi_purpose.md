Purpose of the Report API
The modules created using MetaTrader 5 Report API allow to extend the functional of the MetaTrader 5 platform reports. Using Report API a plugin can read various configurations, databases, information about the current state of clients, etc. This allows making all sorts of reports about the current state of the platform.
Information about the reports generated by the modules is available in two forms:
  * Tabular This mode allows to get information in the form of a table for further processing either through the manager terminal interface (e.g., sorting), or in external applications.
  * HTML This mode allows to get information as a report formatted using HTML. Such reports allow to visualise the information including implementation of various diagrams (SVG).

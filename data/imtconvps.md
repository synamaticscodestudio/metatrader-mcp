IMTConVPS
The IMTConVPS class contains methods for obtaining and changing Sponsored VPS settings:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconvps_release.md "Release") | Delete the current object. |
| [Assign](imtconvps_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconvps_clear.md "Clear") | Clear an object. |
| [Flags](imtconvps_flags.md "Flags") | Get and set additional Sponsored VPS settings. |
| [MQL5Login](imtconvps_mql5login.md "MQL5Login") | Get and set the MQL5 account login used for Sponsored VPS. The method is obsolete and is no longer used. |
| [MQL5Password](imtconvps_mql5password.md "MessageTemplate") | Get and set the MQL5 account password used for Sponsored VPS. The method is obsolete and is no longer used. |
| [RuleAdd](imtconvps_ruleadd.md "RuleAdd") | Add a VPS allocation rule. |
| [RuleUpdate](imtconvps_ruleupdate.md "RuleUpdate") | Edit a VPS allocation rule. |
| [RuleDelete](imtconvps_ruledelete.md "RuleDelete") | Delete a VPS allocation rule. |
| [RuleClear](imtconvps_ruleclear.md "RuleClear") | Clear the list of VPS allocation rules. |
| [RuleShift](imtconvps_ruleshift.md "RuleShift") | Move the VPS allocation rule in the list. |
| [RuleTotal](imtconvps_ruletotal.md "RuleTotal") | Get the number of VPS allocation rules. |
| [RuleNext](imtconvps_rulenext.md "RuleNext") | Getting the VPS allocation rule by index in the list. |
| [GroupAdd](imtconvps_groupadd.md "GroupAdd") | Add a group of accounts in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupUpdate](imtconvps_groupupdate.md "GroupUpdate") | Change a group of accounts in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupDelete](imtconvps_groupdelete.md "GroupDelete") | Remove a group of accounts from the list of groups in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupClear](imtconvps_groupclear.md "GroupClear") | Clear the list of groups in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupShift](imtconvps_groupshift.md "GroupShift") | Move an entry in the list of groups in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupTotal](imtconvps_grouptotal.md "GroupTotal") | Get the number of groups in which the Sponsored VPS is allowed. The method is obsolete and is no longer used. |
| [GroupNext](imtconvps_groupnext.md "GroupNext") | Get the group in which the Sponsored VPS is allowed by its index in the list. The method is obsolete and is no longer used. |

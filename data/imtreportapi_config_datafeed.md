Data Feed Configuration
To get the configurations of data feeds, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [FeederCreate](imtreportapi_feedercreate.md "FeederCreate") | Create an object of the data feed configuration. |
| [FeederModuleCreate](imtreportapi_feedermodulecreate.md "FeederModuleCreate") | Create an object of configuration of the data feed module. |
| [FeederParamCreate](imtreportapi_feederparamcreate.md "FeederParamCreate") | Create an object of the parameter of the data feeds. |
| [FeederTranslateCreate](imtreportapi_feedertranslatecreate.md "FeederTranslateCreate") | Create an object of setup of converting the information transmitted from a data feed. |
| [FeederTotal](imtreportapi_feedertotal.md "FeederTotal") | The total number of data feed configurations available in the platform. |
| [FeederNext](imtreportapi_feedernext.md "FeederNext") | Gets a data feed configuration based on its index. |
| [FeederGet](imtreportapi_feederget.md "FeederGet") | Gets the data feed configuration based on its name. |
| [FeederModuleTotal](imtreportapi_feedermoduletotal.md "FeederModuleTotal") | The total number of configurations of data feed modules (EXE files) available in the platform. |
| [FeederModuleNext](imtreportapi_feedermodulenext.md "FeederModuleNext") | Get the configuration of the data feed module by the index. |
| [FeederModuleGet](imtreportapi_feedermoduleget.md "FeederModuleGet") | Get the configuration of the data feed module by the name. |

Clients
The methods described in this section allow working with the client base on the server:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [UserCreate](php_usercreate.md "UserCreate") | Create an object of a client record. |
| [UserAdd](php_useradd.md "UserAdd") | Create a client account on the server. |
| [UserUpdate](php_userupdate.md "UserUpdate") | Change a client account on the server. |
| [UserDelete](php_userdelete.md "UserDelete") | Delete a client account on the server. |
| [UserGet](php_userget.md "UserGet") | Get information about a client by the login. |
| [UserPasswordCheck](php_userpasswordcheck.md "UserPasswordCheck") | Check a client's password. |
| [UserPasswordChange](php_userpasswordchange.md "UserPasswordChange") | Change a client's password. |
| [UserAccountGet](php_useraccountget.md "UserAccountGet") | Get information about a client's trading account. |
| [UserLogins](php_userlogins.md "UserLogins") | Returns an array of logins of the clients who are included in the specified group. |

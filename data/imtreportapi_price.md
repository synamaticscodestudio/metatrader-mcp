Price Data Functions
Functions described in this section allow requesting price data in the form of ticks and 1-minute bars from the history server.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [TickGet](imtreportapi_tickget.md "TickGet") | Get quotes for a symbol in the specified time range. |
| [TickLast](imtreportapi_ticklast.md "TickLast") | Get the last quote for the specified symbol. |
| [TickStat](imtreportapi_tickstat.md "TickStat") | Get statistical information about quotes for the specified symbol. |
| [TickHistoryGetRaw](imtreportapi_tickhistorygetraw.md "TickHistoryGetRaw") | Get raw quotes (not processed prices in accordance with the configuration of the symbol) for a symbol in the specified time range. |
| [TickHistoryGet](imtreportapi_tickhistoryget.md "TickHistoryGet") | Get the history of quotes processed in accordance with the symbol configuration in the specified time range. |
| [ChartHistoryGet](imtreportapi_charthistoryget.md "ChartHistoryGet") | Request minute bars for a symbol. |

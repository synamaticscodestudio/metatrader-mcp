IMTConLeverageTier
The IMTConLeverageTier class contains method for retrieving and modifying levels in a [floating margin](https://support.metaquotes.net/ru/docs/mt5/platform/administration/leverages) rules:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconleveragetier_release.md "Release") | Delete the current object. |
| [Assign](imtconleveragetier_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconleveragetier_clear.md "Clear") | Clear an object. |
| [RangeFrom](imtconleveragetier_rangefrom.md "RangeFrom") | Get the minimum range value for a level in a floating margin rule. |
| [RangeTo](imtconleveragetier_rangeto.md "RangeTo") | Get the maximum range value for a level in a floating margin rule. |
| [MarginRateInitial](imtconleveragetier_marginrateinitial.md "MarginRateInitial") | Get the initial margin rate for a level in a floating margin rule. |
| [MarginRateMaintenance](imtconleveragetier_marginratemaintenance.md "MarginRateMaintenance") | Get the maintenance margin rate for a level in a floating margin rule. |

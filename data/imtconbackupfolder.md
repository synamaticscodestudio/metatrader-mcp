IMTConBackupFolder
The IMTConBackupFolder interface contains methods for managing user folder backup settings.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconbackupfolder_release.md "Release") | Delete the current object. |
| [Assign](imtconbackupfolder_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconbackupfolder_clear.md "Clear") | Clear an object. |
| [Folder](imtconbackupfolder_folder.md "Folder") | Get and set the path to the backed-up user folder. |
| [Masks](imtconbackupfolder_masks.md "Masks") | Get and set a list or a wildcard pattern of files to back up. |
| [Filter](imtconbackupfolder_filter.md "Filter") | Get and set a list or a wildcard pattern of files excluded from backup. |
| [Flags](imtconbackupfolder_flags.md "Flags") | Get and set additional folder backup settings. |

The IMTConBackupFolder class contains the following enumerations:
| Enumeration | Purpose |
| --- | --- |
| --- | --- |
| [EnBackupFlags](imtconbackupfolder_enum.htm#enbackupflags) | Additional folder backup settings. |

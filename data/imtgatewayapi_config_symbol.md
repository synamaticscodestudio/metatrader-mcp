Configuration of Symbols
The functions described in this section allow to manage symbols, as well as subscribe and unsubscribe from events associated with their change.
The following functions for managing symbols are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [SymbolCreate](imtgatewayapi_symbolcreate.md "SymbolCreate") | Create an object of the symbol configuration. |
| [SymbolSessionCreate](imtgatewayapi_symbolsessioncreate.md "SymbolSessionCreate") | Create an object of configuration of a trading or quoting session of the symbol. |
| [SymbolSubscribe](imtgatewayapi_symbolsubscribe.md "SymbolSubscribe") | Subscribe to events and hooks associated with the configuration of symbols. |
| [SymbolUnsubscribe](imtgatewayapi_symbolunsubscribe.md "SymbolUnsubscribe") | Unsubscribe from events and hooks associated with the configuration of symbols. |
| [SymbolAddPreliminary](imtgatewayapi_symboladdpreliminary.md "SymbolAddPreliminary") | Add or update a preliminary configuration of a symbol. |
| [SymbolUpdate](imtgatewayapi_symbolupdate.md "SymbolUpdate") | Add or update a symbol configuration. |
| [SymbolDelete](imtgatewayapi_symboldelete.md "SymbolDelete") | Delete a symbol configuration by the index or name. |
| [SymbolTotal](imtgatewayapi_symboltotal.md "SymbolTotal") | Get the number of the symbols configurations avaialble for a gateway or a data feed. |
| [SymbolNext](imtgatewayapi_symbolnext.md "SymbolNext") | Get the symbol configuration by the index. |
| [SymbolGet](imtgatewayapi_symbolget.md "SymbolGet") | Get a symbol configuration or an individual configuration of a symbol for a group by the name of the symbol. |

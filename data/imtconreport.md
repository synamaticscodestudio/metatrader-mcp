IMTConReport
The IMTConReport class contains methods for getting and changing configurations of reports:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconreport_release.md "Release") | Delete the current object. |
| [Assign](imtconreport_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconreport_clear.md "Clear") | Clear an object. |
| [Name](imtconreport_name.md "Name") | Get and set the name of a report. |
| [Server](imtconreport_server.md "Server") | Get and set the server ID. |
| [Module](imtconreport_module.md "Module") | Get and set the name of a report module. |
| [Mode](imtconreport_mode.md "Mode") | Get and set the report operation mode. |
| [ParameterAdd](imtconreport_parameteradd.md "ParameterAdd") | Add a report parameter. |
| [ParameterUpdate](imtconreport_parameterupdate.md "ParameterUpdate") | Update the parameter of a report in the list. |
| [ParameterDelete](imtconreport_parameterdelete.md "ParameterDelete") | Delete a report parameter by the index. |
| [ParameterClear](imtconreport_parameterclear.md "ParameterClear") | Clear the list of report parameters. |
| [ParameterShift](imtconreport_parametershift.md "ParameterShift") | Change the position of a report parameter in the list. |
| [ParameterTotal](imtconreport_parametertotal.md "ParameterTotal") | Get the number of report parameters. |
| [ParameterNext](imtconreport_parameternext.md "ParameterNext") | Get parameters of a report by the index. |
| [ParameterGet](imtconreport_parameterget.md "ParameterGet") | Get a report parameter by the name. |

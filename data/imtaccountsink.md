IMTAccountSink
The IMTAccountSink class is designed for working with the events related to changes in the account trading state. The class contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnAccountMarginCallEnter](imtaccountsink_onaccountmargincallenter.md "OnAccountMarginCallEnter") | The event handler for the account entering the Margin Call state. Used only in the Manager API. |
| [OnAccountMarginCallLeave](imtaccountsink_onaccountmargincallleave.md "OnAccountMarginCallLeave") | The event handler for the account exiting the Margin Call state. Used only in the Manager API. |
| [OnAccountStopOutEnter](imtaccountsink_onaccountstopoutenter.md "OnAccountStopOutEnter") | The event handler for the account entering the Stop Out state. Used only in the Manager API. |
| [OnAccountStopOutLeave](imtaccountsink_onaccountstopoutleave.md "OnAccountStopOutLeave") | The event handler for the account exiting the Stop Out state. Used only in the Manager API. |

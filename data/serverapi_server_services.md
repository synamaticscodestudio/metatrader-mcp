Server Services
The MetaTrader 5 Server API provides functions for working with auxiliary services of the server. Using them you can restart the server as well as handle events on the server.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ServerRestart](imtserverapi_serverrestart.md "ServerRestart") | Restart the server on which the plugin is running. |
| [ServerRestartRemote](imtserverapi_serverrestartremote.md "ServerRestartRemote") | Restart the server with the specified identifier. |
| [ServerSubscribe](imtserverapi_serversubscribe.md "ServerSubscribe") | Subscribe to events associated with server events. |
| [ServerUnsubscribe](imtserverapi_serverunsubscribe.md "ServerUnsubscribe") | Undubscribe from events associated with server events. |

Funds and ETFs
MetaTrader 5 enables the automation of fund operations. This includes investor and manager operations, investment management, access to charts and reports: For more details, please read the [MetaTrader 5 Administrator Help](https://support.metaquotes.net/en/docs/mt5/platform/administration/fund_etf).
The functions described in this section enable the management of fund settings:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [FundCreate](imtreportapi_fundcreate.md "FundCreate") | Create a fund configuration object. |
| [FundAccountCreate](imtreportapi_fundaccountcreate.md "FundAccountCreate") | Create a fund manager object. |
| [FundInvestorCreate](imtreportapi_fundinvestorcreate.md "FundInvestorCreate") | Create a fund investor object. |
| [FundTotal](imtreportapi_fundtotal.md "FundTotal") | Get the total number of fund configurations available in the platform. |
| [FundNext](imtreportapi_fundnext.md "FundNext") | Get a fund configuration by index. |
| [FundGet](imtreportapi_fundget.md "FundGet") | Get a fund configuration by name. |

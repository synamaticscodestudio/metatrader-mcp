Data Feeds
The following requests are provided for working with data feed configurations in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/feeder/restart](webapi_feeder_restart.md "Restart") | Restart all data feeds in the platform.+ |
| [/api/feeder/add](webapi_feeder_add.md "Add") | Add and update a data feed configuration in the trading platform. |
| [/api/feeder/delete](webapi_feeder_delete.md "Delete") | Delete a data feed configuration from the trading platform. |
| [/api/feeder/shift](webapi_feeder_shift.md "Shift") | Change the position of a data feed configuration in the list. |
| [/api/feeder/total](webapi_feeder_total.md "Get Total") | Get the total number of data feed configurations existing in the platform. |
| [/api/feeder/next](webapi_feeder_next.md "Get by Index") | Get the configuration of one or more data feeds by index in the list. |
| [/api/feeder/get](webapi_feeder_get.md "Get by Name") | Get a data feed configuration by its name. |
| [/api/feeder/get_all](webapi_feeder_get_all.md "Get All") | Get all existing data feed configurations. |
| [/api/feeder/module/total](webapi_feeder_module_total.md "Get Total Modules") | Get the number of data feed modules available in the platform. |
| [/api/feeder/module/next](webapi_feeder_module_next.md "Get Module by Index") | Get the description of a data feed module by the index in the list. |
| [/api/feeder/module/get](webapi_feeder_module_get.md "Get Module by Name") | Get the description of a data feed module by its name. |

IMTOnline
The IMTOnline class contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtonline_release.md "Release") | Delete the current object. |
| [Assign](imtonline_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtonline_clear.md "Clear") | Clear an object. |
| [SessionID](imtonline_sessionid.md "SessionID") | Get a unique ID of a user connection session. |
| [Login](imtonline_login.md "Login") | Get the login of a user. |
| [Group](imtonline_group.md "Group") | Get the group to which the user is included. |
| [Address](imtonline_address.md "Address") | Get the IP address from which the user connected to the server. |
| [Type](imtonline_type.md "Type") | Get a user connection type. |
| [Build](imtonline_build.md "Build") | Get build number of a terminal used for connection. |
| [Time](imtonline_time.md "Time") | Gets connection date. |
| [ComputerID](imtonline_computerid.md "ComputerID") | Get CID, i.e. the unique ID of the computer from which the user connected to the server. |

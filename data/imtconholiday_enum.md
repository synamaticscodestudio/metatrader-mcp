Enumerations
The [IMTConHoliday](imtconholiday.md "IMTConHoliday") class contains one enumeration.
IMTConHoliday::EnHolidayMode
Possible holiday states are listed in IMTConHoliday::EnHolidayMode.
| ID | Value | Description |
| --- | --- | --- |
| --- | --- | --- |
| HOLIDAY_DISABLED | 0 | The holiday is disabled. |
| HOLIDAY_ENABLED | 1 | The holiday is enabled. |
| HOLIDAY_FIRST |  | Beginning of enumeration. It corresponds to HOLIDAY_DISABLED. |
| HOLIDAY_LAST |  | End of enumeration. It corresponds to HOLIDAY_ENABLED. |

This enumeration is used in the [IMTConHoliday::Mode](imtconholiday_mode.md "Mode") method.

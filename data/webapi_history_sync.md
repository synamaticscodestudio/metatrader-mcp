History Synchronization
The following requests are provided for working with historical data synchronization settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/history_sync/start](webapi_history_sync_start.md "Start") | Start history synchronization in accordance with configuration settings. |
| [/api/history_sync/add](webapi_history_sync_add.md "Add") | Add and update a history synchronization configuration in the trading platform. |
| [/api/history_sync/delete](webapi_history_sync_delete.md "Delete") | Delete a history synchronization configuration from the trading platform. |
| [/api/history_sync/shift](webapi_history_sync_shift.md "Shift") | Change the position of a synchronization configuration in the list. |
| [/api/history_sync/total](webapi_history_sync_total.md "Get Total") | Get the total number of history synchronization configurations existing in the platform. |
| [/api/history_sync/next](webapi_history_sync_next.md "Get by Index") | Get one or more history synchronization configurations by index in the list. |
| [/api/history_sync/get_all](webapi_history_sync_get_all.md "Get All") | Get all existing history synchronization configurations. |

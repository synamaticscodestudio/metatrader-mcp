Deals
Using the functions, you can view a server database of deals. The following functions are available for working with deals:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DealCreate](imtmanagerapi_dealcreate.md "DealCreate") | Create an object of a deal. |
| [DealCreateArray](imtmanagerapi_dealcreatearray.md "DealCreateArray") | Create an object of the array of deals. |
| [DealSubscribe](imtmanagerapi_dealsubscribe.md "DealSubscribe") | Subscribe to events associated with changes in the database of deals. |
| [DealUnsubscribe](imtmanagerapi_dealunsubscribe.md "DealUnsubscribe") | Unsubscribe from the events associated with changes in the database of deals. |
| [DealRequest](imtmanagerapi_dealrequest.md "DealRequest") | Get a deal by a ticket and login. |
| [DealRequestByGroup](imtmanagerapi_dealrequestbygroup.md "DealRequestByGroup") | Receive deals related to a group of accounts. |
| [DealRequestByGroupSymbol](imtmanagerapi_dealrequestbygroupsymbols.md "DealRequestByGroupSymbol") | Get deals by group and symbol. |
| [DealRequestByLogins](imtmanagerapi_dealrequestbylogins.md "DealRequestByLogins") | Receive deals by the list of logins. |
| [DealRequestByLoginsSymbol](imtmanagerapi_dealrequestbyloginssymbol.md "DealRequestByLoginsSymbol") | Get deals by a list of logins and symbol. |
| [DealRequestByTickets](imtmanagerapi_dealrequestbytickets.md "DealRequestByTickets") | Receive deals by the list of tickets. |
| [DealRequestPage](imtmanagerapi_dealrequestpage.md "DealRequestPage") | Get client deals with a paged output. |
| [DealAdd](imtmanagerapi_dealadd.md "DealAdd") | Add a deal to the server database. |
| [DealAddBatch](imtmanagerapi_dealaddbatch.md "DealAddBatch") | Add a batch of deals to the server database. |
| [DealAddBatchArray](imtmanagerapi_dealaddbatcharray.md "DealAddBatchArray") | Add a batch of deals to the server database. |
| [DealUpdate](imtmanagerapi_dealupdate.md "DealUpdate") | Update a deal. |
| [DealUpdateBatch](imtmanagerapi_dealupdatebatch.md "DealUpdateBatch") | Update deals in a server database in bulk. |
| [DealUpdateBatchArray](imtmanagerapi_dealupdatebatcharray.md "DealUpdateBatchArray") | Update deals in a server database in bulk. |
| [DealDelete](imtmanagerapi_dealdelete.md "DealDelete") | Delete a deal. |
| [DealDeleteBatch](imtmanagerapi_dealdeletebatch.md "DealDeleteBatch") | Delete deals from the server database in bulk. |
| [DealPerform](imtmanagerapi_dealperform.md "DealPerform") | Perform a deal on the client's account. This method performs a market buy or sell operation on the account as if it were performed by the client through the terminal. |
| [DealPerformBatch](imtmanagerapi_dealperformbatch.md "DealPerformBatch") | Perform multiple deals on the client's account. This method performs market buy or sell operations on the account as if they were performed by the client through the terminal. |
| [DealPerformBatchArray](imtmanagerapi_dealperformbatcharray.md "DealPerformBatchArray") | Perform multiple deals on the client's account. This method performs market buy or sell operations on the account as if they were performed by the client through the terminal. |

Custom Events Interface IMTCustomSink
The IMTCustomSink interface for handling custom events includes the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [HookManagerCommand](imtcustomsink_hookmanagercommand.md "HookManagerCommand") | A hook of an event of execution of a manager's custom command. |
| [HookWebAPICommand](imtcustomsink_hookwebapicommand.md "HookWebAPICommand") | A hook of an event of execution of a web-client's command. |
| [HookPluginCommand](imtcustomsink_hookplugincommand.md "HookPluginCommand") | Hook for the execution of a custom command transmitted by the plugin from another server within the cluster. |

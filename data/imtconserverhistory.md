IMTConServerHistory
The IMTConServerHistory interface contains methods for managing settings that are specific to History Servers.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconserverhistory_release.md "Release") | Delete the current object. |
| [Assign](imtconserverhistory_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconserverhistory_clear.md "Clear") | Clear an object. |
| [DatafeedsTimeout](imtconserverhistory_datafeedstimeout.md "DatafeedsTimeout") | Get and set timeout of data feeds before switching to other data feeds. |
| [NewsMax](imtconserverhistory_newsmax.md "NewsMax") | Getting and setting the maximum number of news stored on the history server. |

The ECN
MetaTrader 5 API provides access to [built-in ECN](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn) databases. Functions from these section enable the detailed analysis of the current matching queue and history, of operations with liquidity providers and other data.
  * [IMTECNMatching](imtecnmatching.md "IMTECNMatching") — [matching order](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching_history#matching_orders_current) object interface.
  * [IMTECNMatchingArray](imtecnmatchingarray.md "IMTECNMatchingArray") — interface for the array of matching orders.
  * [IMTECNFilling](imtecnfilling.md "IMTECNFilling") — [filling order](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching_history#filling_orders_current) object interface.
  * [IMTECNFillingArray](imtecnfillingarray.md "IMTECNFillingArray") — interface for the array of filling orders.
  * [IMTECNProvider](imtecnprovider.md "IMTECNProvider") — interface for the [provider](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching#providers) via which the requests are executed.
  * [IMTECNProviderArray](imtecnproviderarray.md "IMTECNProviderArray") — interface for an array of providers.
  * [IMTECNHistoryMatching](imtecnhistorymatching.md "IMTECNMatching") — interface for the [matching order](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching_history#matching_orders) object from history.
  * [IMTECNHistoryMatchingArray](imtecnhistorymatchingarray.md "IMTECNMatchingArray") — interface for an array of objects of matching orders from history.
  * [IMTECNHistoryFilling](imtecnhistoryfilling.md "IMTECNFilling") — interface for the [filling order](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching_history#filling_orders) object from history.
  * [IMTECNHistoryFillingArray](imtecnhistoryfillingarray.md "IMTECNFillingArray") — interface for an array of objects of filling orders from history.
  * [IMTECNHistoryDeal](imtecnhistorydeal.md "IMTECNFilling") — interface for the object of a [deal](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching_history#deals) from the ECN history.
  * [IMTECNHistoryDealArray](imtecnhistorydealarray.md "IMTECNMatchingArray") — interface for an array of objects of deals from the ECN history.

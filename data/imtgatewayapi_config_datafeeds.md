The functions described in this section allow to get data feeds configurations:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [FeederCreate](imtgatewayapi_feedercreate.md "FeederCreate") | Create an object of the data feed configuration. |
| [FeederParamCreate](imtgatewayapi_feederparamcreate.md "FeederParamCreate") | Create an object of the parameter of the data feeds. |

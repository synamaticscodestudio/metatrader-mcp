IMTConPlugin
The IMTConPlugin class contains methods for getting and changing plugin configurations:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconplugin_release.md "Release") | Delete the current object. |
| [Assign](imtconplugin_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconplugin_clear.md "Clear") | Clear an object. |
| [Name](imtconplugin_name.md "Name") | Get and set the name of a plugin. |
| [Server](imtconplugin_server.md "Server") | Get and set the server ID. |
| [Module](imtconplugin_module.md "Module") | Get and set the plugin module name. |
| [Mode](imtconplugin_mode.md "Mode") | Get and set the plugin operation mode. |
| [Flags](imtconplugin_flags.md "Flags") | Get ans set the plugin operation flags. |
| [ParameterAdd](imtconplugin_parameteradd.md "ParameterAdd") | Add a plugin parameter. |
| [ParameterUpdate](imtconplugin_parameterupdate.md "ParameterUpdate") | Update the parameter of a plugin in the list. |
| [ParameterDelete](imtconplugin_parameterdelete.md "ParameterDelete") | Delete a plugin parameter by the index. |
| [ParameterClear](imtconplugin_parameterclear.md "ParameterClear") | Clear the list of plugin parameters. |
| [ParameterShift](imtconplugin_parametershift.md "ParameterShift") | Change the position of a plugin parameter in the list. |
| [ParameterTotal](imtconplugin_parametertotal.md "ParameterTotal") | Get the number of plugin parameters. |
| [ParameterNext](imtconplugin_parameternext.md "ParameterNext") | Get the plugin parameters by the index. |
| [ParameterGet](imtconplugin_parameterget.md "ParameterGet") | Get the plugin parameter by the name. |

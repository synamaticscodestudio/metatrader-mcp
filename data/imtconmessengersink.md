IMTConMessengerSink
The IMTConMessengerSink class contains event handlers related to manager configuration changes:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnMessengerAdd](imtconmessengersink_onmessengeradd.md "OnMessengerAdd") | A handler of the the addition of a new messenger configuration. |
| [OnMessengerUpdate](imtconmessengersink_onmessengerupdate.md "OnMessengerUpdate") | A handler of the messenger configuration update event. |
| [OnMessengerDelete](imtconmessengersink_onmessengerdelete.md "OnMessengerDelete") | A handler of the messenger configuration deletion event. |
| [OnMessengerSync](imtconmessengersink_onmessengersync.md "OnMessengerSync") | A handler of the messenger configuration synchronization event. |

Mail Database Functions
Using the functions, you can create and send news, as well as subscribe and unsubscribe from events with changes in the news database.
The following functions are available for working with news:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [NewsCreate](imtmanagerapi_newscreate.md "NewsCreate") | Create an object of a news item. |
| [NewsSubscribe](imtmanagerapi_newssubscribe.md "NewsSubscribe") | Subscribe to events and hooks associated with changes in the news database. |
| [NewsUnsubscribe](imtmanagerapi_newsunsubscribe.md "NewsUnsubscribe") | Undubscribe from events and hooks associated with changes in the news database. |
| [NewsTotal](imtmanagerapi_newstotal.md "NewsTotal") | Get the total number of news items received by the manager. |
| [NewsNext](imtmanagerapi_newsnext.md "NewsNext") | Get the news item received by the manager. |
| [NewsBodyRequest](imtmanagerapi_newsbodyrequest.md "NewsBodyRequest") | Get the body of the news item received by the manager. |
| [NewsSend](imtmanagerapi_newssend.md "NewsSend") | Send news. |

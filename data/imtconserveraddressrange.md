IMTConServerAddressRange
The IMTConServerAddressRange interface contains methods for configuring ranges of IP addresses. The interface is used for configuring the Anti DDoS Proxy Server component, which enables the use of [external Anti-DDoS service providers](https://support.metaquotes.net/en/docs/mt5/platform/administration/admin_network/network_anti_ddos).
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconserveraddressrange_release.md "Release") | Delete the current object. |
| [Assign](imtconserveraddressrange_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconserveraddressrange_clear.md "Clear") | Clear an object. |
| [From](imtconserveraddressrange_from.md "From") | Get and set the beginning of the range of IP addresses. |
| [To](imtconserveraddressrange_to.md "To") | Get and set the end of the range of IP addresses. |

Firewall
The following requests are provided for retrieving platform firewall configurations:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/firewall/add](webapi_firewall_add.md "Add") | Add and update a trading platform firewall rule. |
| [/api/firewall/delete](webapi_firewall_delete.md "Delete") | Delete a trading platform firewall rule. |
| [/api/firewall/shift](webapi_firewall_shift.md "Shift") | Change the position of a firewall rule in the list. |
| [/api/firewall/total](webapi_firewall_total.md "Get Total") | Get the total number of firewall rules existing in the platform. |
| [/api/firewall/next](webapi_firewall_next.md "Get by Index") | Get one or more firewall rules by index in the list. |
| [/api/firewall/get_all](webapi_firewall_get_all.md "Get All") | Get all existing firewall rules. |

IMTConKYCGroup
The IMTConKYCGroup method contains classes for getting and changing group settings in KYC providers:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconkycgroup_release.md "Release") | Delete the current object. |
| [Assign](imtconkycgroup_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconkycgroup_clear.md "Clear") | Clear an object. |
| [Group](imtconkycgroup_group.md "Group") | Get and set the group of accounts for which the KYC provider is used. |

IMTConGatewayTranslate
The IMTConGatewayTranslate class contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtcongatewaytranslate_release.md "Release") | Delete the current object. |
| [Assign](imtcongatewaytranslate_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtcongatewaytranslate_clear.md "Clear") | Clear an object. |
| [Source](imtcongatewaytranslate_source.md "Source") | Get and set the symbol name in the data feed, to which the gateway connects. |
| [Symbol](imtcongatewaytranslate_symbol.md "Symbol") | Get and set the target symbol name in the trading platform. |
| [BidMarkup](imtcongatewaytranslate_bidmarkup.md "BidMarkup") | Gets and sets a markup value for the Bid price received for a symbol from the data source to which the gateway connects. |
| [AskMarkup](imtcongatewaytranslate_askmarkup.md "AskMarkup") | Gets and sets a markup value for the Ask price received for a symbol from the data source to which the gateway connects. |
| [Digits](imtcongatewaytranslate_digits.md "Digits") | Get the number of digits after the decimal point in the price of the symbol that receives quotes. |

Detailed information about [symbol and price translation features](gatewayapi_translation.md "Symbol and Price Translation") is available in the Gateway API section.

Configuration of Groups
To get groups configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [GroupCreate](imtreportapi_groupcreate.md "GroupCreate") | Create an object of the group configuration. |
| [GroupSymbolCreate](imtreportapi_groupsymbolcreate.md "GroupSymbolCreate") | Create an object of symbol configuration for a group. |
| [GroupCommissionCreate](imtreportapi_groupcommissioncreate.md "GroupCommissionCreate") | Create an object of commission configuration for a group. |
| [GroupTierCreate](imtreportapi_grouptiercreate.md "GroupTierCreate") | Create an object of commission range configuration for a group. |
| [GroupTotal](imtreportapi_grouptotal.md "GroupTotal") | The total number of group configurations available in the platform. |
| [GroupNext](imtreportapi_groupnext.md "GroupNext") | Get the group configuration by the index. |
| [GroupGet](imtreportapi_groupget.md "GroupGet") | Get the group configuration by the name. |
| [GroupGetLight](imtreportapi_groupgetlight.md "GroupGetLight") | Get an eased configuration of a group. |

Enumerations
The [IMTConReport](imtconreport.md "IMTConReport") class contains one enumeration:
IMTConReport::EnReportMode
Modes of report operation are listed in IMTConReport::EnReportMode.
| ID | Value | Description |
| --- | --- | --- |
| --- | --- | --- |
| REPORT_DISABLED | 0 | The report is disabled. |
| REPORT_ENABLED | 1 | The report is enabled. |
| REPORT_FIRST |  | Beginning of enumeration. It corresponds to REPORT_DISABLED. |
| REPORT_LAST |  | End of enumeration. It corresponds to REPORT_ENABLED. |

This enumeration is used in the [IMTConReport::Mode](imtconreport_mode.md "Mode") method.

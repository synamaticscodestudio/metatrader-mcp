Daily Report Functions
Functions described in this section allow requesting [daily reports](trading_daily.md "Daily Reports") of users from the server. The following functions for requesting daily reports are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DailyCreate](imtmanagerapi_dailycreate.md "DailyCreate") | Create an object of a daily report. |
| [DailyCreateArray](imtmanagerapi_dailycreatearray.md "DailyCreateArray") | Create an object of the array of daily reports. |
| [DailyRequest](imtmanagerapi_dailyrequest.md "DailyRequest") | Get an array of daily reports by the login and date range. |
| [DailyRequestByLogins](imtmanagerapi_dailyrequestbylogins.md "DailyRequestByLogins") | Get an array of daily reports by the list of logins and date range. |
| [DailyRequestByGroup](imtmanagerapi_dailyrequestbygroup.md "DailyRequestByGroup") | Get an array of daily reports by groups and date range. |
| [DailyRequestLight](imtmanagerapi_dailyrequestlight.md "DailyRequestLight") | Get an array of lightweight daily reports by the login and date range. Unlike full daily reports, lightweight reports do not include open client orders and positions. |
| [DailyRequestLightByLogins](imtmanagerapi_dailyrequestlightbylogins.md "DailyRequestLightByLogins") | Get an array of lightweight daily reports by the list of logins and date range. |
| [DailyRequestLightByGroup](imtmanagerapi_dailyrequestlightbygroup.md "DailyRequestLightByGroup") | Get an array of lightweight daily reports by groups and date range. |

Manage Logging
Implementation of the Web API protocol in PHP includes the option of logging activities of a web client. This section describes the methods for managing logging:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [SetLoggerIsWrite](php_setloggeriswrite.md "SetLoggerIsWrite") | Enable/disable logging of the Web API. |
| [SetLoggerFilePath](php_setloggerfilepath.md "SetLoggerFilePath") | Set the path to the folder where you want to place the log files. |
| [SetLoggerFilePrefix](php_setloggerfileprefix.md "SetLoggerFilePrefix") | Set a prefix for naming log files. |
| [SetLoggerWriteDebug](php_setloggerwritedebug.md "SetLoggerWriteDebug") | Enable/disable logging of debug messages of the Web API. |

Holidays
The following requests are provided for receiving platform holiday settings:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/holiday/add](webapi_holiday_add.md "Add") | Add and update a holiday configuration in the trading platform. |
| [/api/holiday/delete](webapi_holiday_delete.md "Delete") | Delete a holiday configuration from the trading platform. |
| [/api/holiday/shift](webapi_holiday_shift.md "Shift") | Change the position of a holiday configuration in the list. |
| [/api/holiday/total](webapi_holiday_total.md "Get Total") | Get the total number of holiday configurations existing in the platform. |
| [/api/holiday/next](webapi_holiday_next.md "Get by Index") | Get a holiday configuration description by the index in the list. |
| [/api/holiday/get_all](webapi_holiday_get_all.md "Get All") | Get all existing holiday configurations. |

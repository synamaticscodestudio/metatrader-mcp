Configuration of Managers
To get managers configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ManagerCreate](imtreportapi_managercreate.md "ManagerCreate") | Create a manager configuration object. |
| [ManagerAccessCreate](imtreportapi_manageraccesscreate.md "ManagerAccessCreate") | Create an object of configuration of an access list by IP addresses for a manager. |
| [ManagerReportCreate](imtreportapi_managerreportcreate.md "ManagerReportCreate") | Create an object for the manager's permission to access the report. |
| [ManagerCurrent](imtreportapi_managercurrent.md "ManagerCurrent") | Get the configuration of the manager who requested report generation from the manager terminal. |
| [ManagerTotal](imtreportapi_managertotal.md "ManagerTotal") | The total number of manager configurations available in the platform. |
| [ManagerNext](imtreportapi_managernext.md "ManagerNext") | Get a manager configuration with the specified index. |
| [ManagerGet](imtreportapi_managerget.md "ManagerGet") | Get a manager configuration with the specified login. |

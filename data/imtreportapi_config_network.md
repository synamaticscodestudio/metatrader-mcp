Network Configuration
To get the platform components configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [NetServerCreate](imtreportapi_netservercreate.md "NetServerCreate") | Create an object of configuration of the platform components. |
| [NetServerRangeCreate](imtreportapi_netserverrangecreate.md "NetServerRangeCreate") | Create an object of the range of orders, deals or accounts. |
| [NetServerTotal](imtreportapi_netservertotal.md "NetServerTotal") | The total number of server configurations available in the platform. |
| [NetServerNext](imtreportapi_netservernext.md "NetServerNext") | Get a server configuration by the index. |
| [NetServerGet](imtreportapi_netserverget.md "NetServerGet") | Get a server configuration by the ID. |

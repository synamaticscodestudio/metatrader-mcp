IMTConVPSRule
The IMTConVPSRule class contains methods for obtaining and changing rules in Sponsored VPS settings:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconvpsrule_release.md "Release") | Delete the current object. |
| [Assign](imtconvpsrule_assign.md "Assign") | Assign the passed object to the current one. |
| [Clear](imtconvpsrule_clear.md "Clear") | Clear an object. |
| [Enabled](imtconvpsrule_enabled.md "Enabled") | Get the state of the VPS allocation rule. |
| [Name](imtconvpsrule_name.md "Name") | Get the name of the VPS allocation rule. |
| [ConditionAdd](imtconvpsrule_conditionadd.md "ConditionAdd") | Add a condition to the VPS allocation rule. |
| [ConditionUpdate](imtconvpsrule_conditionupdate.md "ConditionUpdate") | Edit a condition in the VPS allocation rule. |
| [ConditionDelete](imtconvpsrule_conditiondelete.md "ConditionDelete") | Delete a condition from the VPS allocation rule. |
| [ConditionClear](imtconvpsrule_conditionclear.md "ConditionClear") | Clear the list of conditions in the VPS allocation rule. |
| [ConditionShift](imtconvpsrule_conditionshift.md "ConditionShift") | Move a condition in the list in the VPS allocation rule. |
| [ConditionTotal](imtconvpsrule_conditiontotal.md "ConditionTotal") | Get the number of conditions in the VPS allocation rule. |
| [ConditionNext](imtconvpsrule_conditionnext.md "ConditionNext") | Get a condition in the VPS allocation rule by index. |

Configuration of Symbols
Use these functions to obtain symbol settings, as well to subscribe and unsubscribe from events associated with their change.
The following functions are available for working with symbol settings:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [SymbolCreate](imtmanagerapi_symbolcreate.md "SymbolCreate") | Create an object of the symbol configuration. |
| [SymbolCreateArray](imtmanagerapi_symbolcreatearray.md "SymbolCreateArray") | Create a symbols array object. |
| [SymbolSessionCreate](imtmanagerapi_symbolsessioncreate.md "SymbolSessionCreate") | Create an object of configuration of a trading or quoting session of the symbol. |
| [SymbolSubscribe](imtmanagerapi_symbolsubscribe.md "SymbolSubscribe") | Subscribe to events associated with the configuration of symbols. |
| [SymbolUnsubscribe](imtmanagerapi_symbolunsubscribe.md "SymbolUnsubscribe") | Unsubscribe from events associated with the configuration of symbols. |
| [SymbolUpdate](imtmanagerapi_symbolupdate.md "SymbolUpdate") | Update symbol configuration. |
| [SymbolUpdateBatch](imtmanagerapi_symbolupdatebatch.md "SymbolUpdateBatch") | Update multiple symbol configurations. |
| [SymbolTotal](imtmanagerapi_symboltotal.md "SymbolTotal") | The total number of symbol configurations available in the platform. |
| [SymbolNext](imtmanagerapi_symbolnext.md "SymbolNext") | Get the symbol configuration by the index. |
| [SymbolGet](imtmanagerapi_symbolget.md "SymbolGet") | Get a symbol configuration or an individual configuration of a symbol for a group by the name of the symbol. |
| [SymbolRequest](imtmanagerapi_symbolrequest.md "SymbolRequest") | Request a symbol configuration from a server by the name. |
| [SymbolRequestArray](imtmanagerapi_symbolrequestarray.md "SymbolRequestArray") | Request from the server an array of symbols by mask. |
| [SymbolExist](imtmanagerapi_symbolexist.md "SymbolExist") | Check the availability of a symbol for a specified [group](config_group.md "Groups") of clients. |

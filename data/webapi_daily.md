Daily Reports
The following requests are provided for operations with daily reports in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/daily/get](webapi_daily_get.md "Get") | Get daily reports by the date range and login. |
| [/api/daily/get_batch](webapi_daily_get_batch.md "Get Multiple") | Retrieve daily reports in batch for a list of user logins. |
| [/api/daily/get_page](webapi_daily_get_page.md "Get Paged") | Retrieve daily reports in batch for a list of user logins with pagination. |
| [/api/daily/get_light](webapi_daily_get_light.md "Get Light") | Get lightweight versions of daily reports by the date range and login. |
| [/api/daily/get_light_batch](webapi_daily_get_light_batch.md "Get Multiple Light") | Retrieve lightweight daily reports in batch for a list of user logins. |
| [/api/daily/get_light_page](webapi_daily_get_light_page.md "Get Light Paged") | Retrieve lightweight daily reports in batch for a list of user logins with pagination. |

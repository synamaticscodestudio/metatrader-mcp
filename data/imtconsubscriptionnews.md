IMTConSubscriptionNews
The IMTConSubscriptionNews class provides method for obtaining and editing [news settings](https://support.metaquotes.net/en/docs/mt5/platform/administration/subscriptions/subscriptions_news) in subscription configurations:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconsubscriptionnews_release.md "Release") | Delete the current object. |
| [Assign](imtconsubscriptionnews_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconsubscriptionnews_clear.md "Clear") | Clear an object. |
| [Category](imtconsubscriptionnews_category.md "Category") | Get and set news categories available by subscription. |
| [Language](imtconsubscriptionnews_language.md "Language") | Get and set the language of news available by subscription. |

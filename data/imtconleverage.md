IMTConLeverage
The IMTConLeverage class contains methods for retrieving and updating [floating margin](https://support.metaquotes.net/en/docs/mt5/platform/administration/leverages) configurations:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconleverage_release.md "Release") | Delete the current object. |
| [Assign](imtconleverage_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconleverage_clear.md "Clear") | Clear an object. |
| [Name](imtconleverage_name.md "Name") | Get and set the name for the floating margin configuration. |
| [RuleAdd](imtconleverage_ruleadd.md "RuleAdd") | Add a rule to a floating margin configuration. |
| [RuleUpdate](imtconleverage_ruleupdate.md "RuleUpdate") | Update a rule in a floating margin configuration. |
| [RuleDelete](imtconleverage_ruledelete.md "RuleDelete") | Delete a rule from a floating margin configuration. |
| [RuleClear](imtconleverage_ruleclear.md "RuleClear") | Clear the list of rules in a floating margin configuration. |
| [RuleShift](imtconleverage_ruleshift.md "RuleShift") | Move a rule in a floating margin configuration. |
| [RuleTotal](imtconleverage_ruletotal.md "RuleTotal") | Get the number of rules in a floating margin configuration. |
| [RuleNext](imtconleverage_rulenext.md "RuleNext") | Get a rule from a floating margin configuration by index. |
| [RuleGet](imtconleverage_ruleget.md "CountryNext") | Get a rule from a floating margin configuration by name. |

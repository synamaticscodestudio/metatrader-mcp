IMTConFundSink
The IMTConFundSink class contains event handlers for fund configuration changes:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnFundAdd](imtconfundsink_onfundadd.md "OnFundAdd") | Event handler for adding of a new fund configuration. |
| [OnFundUpdate](imtconfundsink_onfundupdate.md "OnFundUpdate") | Event handler for fund configuration update. |
| [OnFundSync](imtconfundsink_onfundsync.md "OnFundSync") | Event handler for fund configuration deletion. |
| [OnFundSync](imtconfundsink_onfundsync.md "OnFundSync") | Event handler for the synchronization of fund configurations. |

IMTReportChart
The IMTReportChart interface is used for working with charts. It contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtreportchart_release.md "Release") | Delete the current object. |
| [Assign](imtreportchart_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtreportchart_clear.md "Clear") | Clear an object. |
| [Title](imtreportchart_title.md "Title") | Get and set a chart header. |
| [Type](imtreportchart_type.md "Type") | Get and set a chart type. |
| [Digits](imtreportchart_digits.md "Digits") | Get and set the number of decimal places for formatting the values shown in a chart. |
| [Flags](imtreportchart_flags.md "Flags") | Get and set a chart flags. |
| [BarHeight](imtreportchart_barheight.md "BarHeight") | Get and set the chart bar height. |
| [PieceTooltip](imtreportchart_piecetooltip.md "PieceTooltip") | Get and set a pie chart element tooltip. |
| [PieceDescription](imtreportchart_piecedescription.md "PieceDescription") | Get and set a pie chart element description. |
| [SeriesClear](imtreportchart_seriesclear.md "SeriesClear") | Delete all chart series. |
| [SeriesAdd](imtreportchart_seriesadd.md "SeriesAdd") | Add data series to a chart. |
| [SeriesAddCopy](imtreportchart_seriesaddcopy.md "SeriesAddCopy") | Add a data series copy to a chart. |
| [SeriesDelete](imtreportchart_seriesdelete.md "SeriesDelete") | Delete an object of a data series by its position. |
| [SeriesDetach](imtreportchart_seriesdetach.md "SeriesDetach") | Detach a data series object from a chart. |
| [SeriesUpdate](imtreportchart_seriesupdate.md "SeriesUpdate") | Change a data series at the specified position of a chart. |
| [SeriesShift](imtreportchart_seriesshift.md "SeriesShift") | Change data series position in a chart. |
| [SeriesTotal](imtreportchart_seriestotal.md "SeriesTotal") | Get a number of data series objects in a chart. |
| [SeriesNext](imtreportchart_seriesnext.md "SeriesNext") | Get a data series object by its position. |

The IMTReportChart interface contains the following enumerations:
| Interface | Purpose |
| --- | --- |
| --- | --- |
| [EnChartType](imtreportchart_enum.htm#encharttype) | Types of charts. |
| [EnChartFlags](imtreportchart_enum.htm#enchartflags) | Charts display flags. |

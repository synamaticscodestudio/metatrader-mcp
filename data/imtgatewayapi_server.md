Server
The functions described in this section allow to run and stop Gateway API server port for the platform connection and get the data on the current address and the port where Gateway API server port is launched.
The following functions are provided:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [Start](imtgatewayapi_start.md "Start") | Gateway API server port launch. |
| [Stop](imtgatewayapi_stop.md "Stop") | Gateway API server port stop. |
| [ServerIP](imtgatewayapi_serverip.md "ServerIP") | Get the IP address on which the Gateway API server port is running. |
| [ServerPort](imtgatewayapi_serverport.md "ServerPort") | Get the number of the port used for the connection to Gateway API. |
| [ServerConnections](imtgatewayapi_serverconnections.md "ServerConnections") | Get the number of the current connections to the Gateway API server port. |

How to perform request confirmation
When responding to any type of request, the following fields should be filled in IMTConfirm:
  * [IMTConfirm::ID](imtconfirm_id.md "ID") � the ID of the request, in response to which the confirmation will be formed ([IMTRequest::ID](imtrequest_id.md "ID")).
  * [IMTConfirm::Retcode](imtconfirm_retcode.md "Retcode") � response code for the request.
  * [IMTConfirm::Comment](imtconfirm_comment.md "Comment") � an optional field. If a comment is specified, it will be shown to the client in the trading dialog instead of the standard message corresponding to the response code.

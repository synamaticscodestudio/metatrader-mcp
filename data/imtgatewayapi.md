Main Interface IMTGatewayAPI
Main interaction with API is carried out through the IMTGatewayAPI interface. [Exported functions](gatewayapi_exported.md "Exported Functions") and [factory](cmtgatewayapifactory.md "CMTGatewayAPIFactory") must be used to get the access to the interface. All IMTGatewayAPI interface is fully thread safe. Its methods can be called at any time from any commands execution thread.

Configuration of Plugins
In addition to creating plugins, the MetaTrader 5 Server API allows managing plugin configurations in the trading platform and process events of their configuration changes.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [PluginCreate](imtserverapi_plugincreate.md "PluginCreate") | Create an object of the plugin configuration. |
| [PluginModuleCreate](imtserverapi_pluginmodulecreate.md "PluginModuleCreate") | Create an object of the plugin module configuration. |
| [PluginParamCreate](imtserverapi_pluginparamcreate.md "PluginParamCreate") | Create an object of the plugin parameter. |
| [PluginSubscribe](imtserverapi_pluginsubscribe.md "PluginSubscribe") | Subscribe to events and hooks associated with the configuration of plugins. |
| [PluginUnsubscribe](imtserverapi_pluginunsubscribe.md "PluginUnsubscribe") | Unsubscribe from events and hooks associated with the configuration of plugins. |
| [PluginCurrent](imtserverapi_plugincurrent.md "PluginCurrent") | Get the configuration of the current plugin. |
| [PluginAdd](imtserverapi_pluginadd.md "PluginAdd") | Add or update a plugin configuration. |
| [PluginDelete](imtserverapi_plugindelete.md "PluginDelete") | Delete a plugin configuration by the name or index |
| [PluginShift](imtserverapi_pluginshift.md "PluginShift") | Changes the position of a plugin configuration in the list. |
| [PluginTotal](imtserverapi_plugintotal.md "PluginTotal") | The total number of plugin configurations available in the platform. |
| [PluginNext](imtserverapi_pluginnext.md "PluginNext") | Get the plugin configuration by the index. |
| [PluginGet](imtserverapi_pluginget.md "PluginGet") | Get the plugin configuration by the name. |
| [PluginModuleTotal](imtserverapi_pluginmoduletotal.md "PluginModuleTotal") | The total number of configurations of plugin modules (DLL files) available in the platform. |
| [PluginModuleNext](imtserverapi_pluginmodulenext.md "PluginModuleNext") | Get a plugin module by the index. |
| [PluginModuleGet](imtserverapi_pluginmoduleget.md "PluginModuleGet") | Get the plugin module configuration by the name. |

Connect/Disconnect
The methods described in this section allow you to connect and disconnect a Web client from the trade server.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Connect](php_connect.md "Connect") | Connect a Web client to a trade server. |
| [Disconnect](php_disconnect.md "Disconnect") | Disconnect a Web client from a trade server. |
| [IsConnected](php_isconnected.md "IsConnected") | Check the connection status of the Web client to the trade server. |
| [Ping](php_ping.md "Ping") | Send empty packets to the server to maintain connection. |

Configuration of Groups
Functions described in this section allow to manage groups, as well subscribe and unsubscribe from events associated with their change.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [GroupCreate](imtserverapi_groupcreate.md "GroupCreate") | Create an object of the group configuration. |
| [GroupSymbolCreate](imtserverapi_groupsymbolcreate.md "GroupSymbolCreate") | Create an object of symbol configuration for a group. |
| [GroupCommissionCreate](imtserverapi_groupcommissioncreate.md "GroupCommissionCreate") | Create an object of commission configuration for a group. |
| [GroupTierCreate](imtserverapi_grouptiercreate.md "GroupTierCreate") | Create an object of commission range configuration for a group. |
| [GroupSubscribe](imtserverapi_groupsubscribe.md "GroupSubscribe") | Subscribe to events and hooks associated with the groups configuration. |
| [GroupUnsubscribe](imtserverapi_groupunsubscribe.md "GroupUnsubscribe") | Unsubscribe from events and hooks associated with the groups configuration. |
| [GroupAdd](imtserverapi_groupadd.md "GroupAdd") | Adds or updates a group configuration. |
| [GroupDelete](imtserverapi_groupdelete.md "GroupDelete") | Delete a group configuration by the name or index |
| [GroupShift](imtserverapi_groupshift.md "GroupShift") | Changes the position of a group configuration in the list. |
| [GroupTotal](imtserverapi_grouptotal.md "GroupTotal") | The total number of group configurations available in the platform. |
| [GroupNext](imtserverapi_groupnext.md "GroupNext") | Get the group configuration by the index. |
| [GroupGet](imtserverapi_groupget.md "GroupGet") | Get the group configuration by the name. |

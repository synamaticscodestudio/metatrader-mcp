Data Cache Interfaces
The interfaces described in this section are used for working with the [data cache](imtreportapi_cache.md "Data cache"):
  * [IMTReportCache](imtreportcache.md "IMTReportCache") contains methods for working with the cache, i.e. for data reading and writing.
  * [IMTReportCacheValue](imtreportcachevalue.md "IMTReportCacheValue") creates methods for working with the cache value and iterating between values.
  * [IMTReportCacheKeySet](imtreportcachekeyset.md "IMTReportCacheKeySet") creates methods for working with the sets of data cache keys.

SMTSearch
This class is designed for sorting and searching for elements in data arrays. It contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Insert](smtsearch_insert.md "Insert") | Insert an element in a pre-sorted array without disturbing the sort order. |
| [QuickSort](smtsearch_quicksort.md "QuickSort") | Sort an array using the sort function passed. |
| [Search](smtsearch_search.md "Search") | Search in an array the array element that matches the search key. |
| [SearchGreatOrEq](smtsearch_searchgreatoreq.md "SearchGreatOrEq") | Search in an array the first element greater than or equal to the search key. |
| [SearchGreater](smtsearch_searchgreater.md "SearchGreater") | Search in an array the first element greater than the search key. |
| [SearchLessOrEq](smtsearch_searchlessoreq.md "SearchLessOrEq") | Search in an array the first element less than or equal to the search key. |
| [SearchLess](smtsearch_searchless.md "SearchLess") | Search in an array the first element less than the search key. |
| [SearchLeft](smtsearch_searchleft.md "SearchLeft") | Search in an array the first element equal to the search key. |
| [SearchRight](smtsearch_searchright.md "SearchRight") | Search in an array the last element equal to the search key. |

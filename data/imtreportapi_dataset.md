Data set
Data sets are used for preparing source data for the creation of [dashboards](reportapi_dashboards.md "Dashboards") and [tabular reports](reportapi_tables.md "Tabular Reports"). Information in a set is stored as a table with a set of columns and rows and a special summary row.
The following functions are available for working with data sets:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DatasetAppend](imtreportapi_datasetappend.md "DatasetAppend") | Add a data set to a dashboard. |
| [DatasetClear](imtreportapi_datasetclear.md "DatasetClear") | Delete all data sets from a dashboard. |
| [DatasetDelete](imtreportapi_datasetdelete.md "DatasetDelete") | Delete a data set from a dashboard. |
| [DatasetTotal](imtreportapi_datasettotal.md "DatasetTotal") | Get the number of data sets in a dashboard. |
| [DatasetNext](imtreportapi_datasetnext.md "DatasetNext") | Get the data set description based in the index. |
| [DatasetRequestCreate](imtreportapi_datasetrequestcreate.md "DatasetRequestCreate") | Create a data request object. |

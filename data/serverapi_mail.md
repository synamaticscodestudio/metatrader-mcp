Mail Database
The MetaTrader 5 Server API provides functions for working with the internal mail system of the platform. Using them you can receive and send emails, as well as handle events associated with changes in the mail database of the server.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [MailCreate](imtserverapi_mailcreate.md "MailCreate") | Create a message in the internal mail system. |
| [MailSubscribe](imtserverapi_mailsubscribe.md "MailSubscribe") | Subscribe to events and hooks associated with changes in the mail database. |
| [MailUnsubscribe](imtserverapi_mailunsubscribe.md "MailUnsubscribe") | Undubscribe from events and hooks associated with changes in the mail database. |
| [MailSend](imtserverapi_mailsend.md "MailSend") | Send emails via the internal mail system. |

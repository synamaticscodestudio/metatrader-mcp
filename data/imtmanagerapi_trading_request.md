Trade Requests
Functions described in this section allow working with the server trade requests queue. They allow to get existing trade requests and subscribe to events associated with changes in the queue of requests.
The following functions are available for working with trade requests:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [RequestCreate](imtmanagerapi_requestcreate.md "RequestCreate") | Create an object of a trade request. |
| [RequestCreateArray](imtmanagerapi_requestcreatearray.md "RequestCreateArray") | Create an object of the array of trade requests. |
| [RequestSubscribe](imtmanagerapi_requestsubscribe.md "RequestSubscribe") | Subscribe to events associated with trade requests queue changes. |
| [RequestUnsubscribe](imtmanagerapi_requestunsubscribe.md "RequestUnsubscribe") | Unsubscribe from events associated with requests queue changes. |
| [RequestTotal](imtmanagerapi_requesttotal.md "RequestTotal") | Get the total amount of trade requests in a requests queue. |
| [RequestNext](imtmanagerapi_requestnext.md "RequestNext") | Get a trade request by a queue position. |
| [RequestGet](imtmanagerapi_requestget.md "RequestGet") | Get a trade request by ID. |
| [RequestGetAll](imtmanagerapi_requestgetall.md "RequestGetAll") | Get all the trade requests in a queue. |

IMTCommentSink
The IMTCommentSink class is designed for working with the comments database events. The class contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnCommentAdd](imtcommentsink_oncommentadd.md "OnCommentAdd") | New comment adding event handler. |
| [OnCommentUpdate](imtcommentsink_oncommentupdate.md "OnCommentUpdate") | Comment update event handler. |
| [OnCommentDelete](imtcommentsink_oncommentdelete.md "OnCommentDelete") | Comment deletion event handler. |

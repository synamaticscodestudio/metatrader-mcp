IMTConAutoParam
The IMTConAutoParam class contains methods for working with [the parameters of actions](https://support.metaquotes.net/en/docs/mt5/platform/administration/automation/automation_action) performed upon the triggering of automation tasks:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconautoparam_release.md "Release") | Delete the current object. |
| [Assign](imtconautoparam_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconautoparam_clear.md "Clear") | Clear an object. |
| [Param](imtconautoparam_condition.md "Condition") | Get and set the automation action parameter type. |
| [ValueType](imtconautoparam_valuetype.md "ValueType") | Get the action parameter value type. |
| [ValueInt](imtconautoparam_valueint.md "ValueInt") | Get and set a parameter value of INT type. |
| [ValueUInt](imtconautoparam_valueuint.md "ValueUInt") | Get and set a parameter value of UINT type. |
| [ValueDouble](imtconautoparam_valuedouble.md "ValueDouble") | Get and set a parameter value of double type. |
| [ValueString](imtconautoparam_valuestring.md "ValueString") | Get and set a parameter value of the string type. |
| [ValueColor](imtconautoparam_valuecolor.md "ValueColor") | Get and set a parameter value of the colorref type. |
| [ValueMoney](imtconautoparam_valuemoney.md "ValueMoney") | Get and set the value of the parameter that expresses the amount of money. |
| [ValueVolume](imtconautoparam_valuevolume.md "ValueVolume") | Get and set the value of the parameter that expresses the volume. |
| [ValueDatetime](imtconautoparam_valuedatetime.md "ValueDatetime") | Get and set the value of the parameter that expresses date and time. |
| [ValueLeverage](imtconautoparam_valueleverage.md "ValueLeverage") | Get and set the value of the parameter that expresses the leverage. |
| [ValueBool](imtconautoparam_valuebool.md "ValueBool") | Get and set a parameter value of bool type. |
| [ValueTime](imtconautoparam_valuetime.md "ValueTime") | Get and set the value of the parameter that expresses the time. |
| [ValueDate](imtconautoparam_valuedate.md "ValueDate") | Get and set the value of the parameter that expresses the date. |
| [ValuePercent](imtconautoparam_valuepercent.md "ValuePercent") | Get and set the value of the parameter that expresses the percentage. |
| [ValueLanguage](imtconautoparam_valuelanguage.md "ValueLanguage") | Get and set the value of the parameter that expresses the language. |
| [ValueServer](imtconautoparam_valueserver.md "ValueServer") | Get and set the value of the parameter that expresses the server ID. |
| [ValueHTML](imtconautoparam_valuehtml.md "ValueHTML") | Get and set the parameter value of type "HTML content". |

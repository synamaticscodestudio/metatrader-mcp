Clients
The methods described in this section allow working with the client base on the server:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [UserAdd](net_useradd.md "UserAdd") | Create a client account on the server. |
| [UserUpdate](net_userupdate.md "UserUpdate") | Change a client account on the server. |
| [UserDelete](net_userdelete.md "UserDelete") | Delete a client account on the server. |
| [UserGet](net_userget.md "UserGet") | Get information about a client by the login. |
| [UserPasswordCheck](net_userpasswordcheck.md "UserPasswordCheck") | Check a client's password. |
| [UserPasswordChange](net_userpasswordchange.md "UserPasswordChange") | Change a client's password. |
| [UserAccountGet](net_useraccountget.md "UserAccountGet") | Get information about a client's trading account. |
| [UserLogins](net_userlogins.md "UserLogins") | Returns a list of logins of the clients who are included in the specified group. |

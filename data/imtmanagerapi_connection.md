Connecting to the Server
This section contains descriptions of methods for connecting and disconnecting an application from the server. The application is connected to the platform in one of the [pumping mode](imtmanagerapi_enpumpmodes.md "Pumping Mode"), which indicates what information the application will continuously receive from the server.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Connect](imtmanagerapi_connect.md "Connect") | Connect to the trading platform. |
| [Disconnect](imtmanagerapi_disconnect.md "Disconnect") | Disconnect from the trading platform. |
| [Subscribe](imtmanagerapi_subscribe.md "Subscribe") | Subscribe to common events of the [IMTManagerAPI](imtmanagerapi.md "Manager Interface") interface. |
| [Unsubscribe](imtmanagerapi_unsubscribe.md "Unsubscribe") | Unsubscribe from common events of the [IMTManagerAPI](imtmanagerapi.md "Manager Interface") interface. |
| [ProxySet](imtmanagerapi_proxyset.md "ProxySet") | Set parameters of connection to the trading platform through a proxy server. |

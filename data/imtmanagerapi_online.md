Functions
The functions allow you to receive data concerning online connections to the trade server. The following functions are provided:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [OnlineCreate](imtmanagerapi_onlinecreate.md "OnlineCreate") | Create connection record object. |
| [OnlineCreateArray](imtmanagerapi_onlinecreatearray.md "OnlineCreateArray") | Create connection record array object. |
| [OnlineTotal](imtmanagerapi_onlinetotal.md "OnlineTotal") | Get the total amount of the current connections to the trade server. |
| [OnlineNext](imtmanagerapi_onlinenext.md "OnlineNext") | Get connection record by index. |
| [OnlineGet](imtmanagerapi_onlineget.md "OnlineGet") | Get connection record by login. |
| [OnlineDisconnect](imtmanagerapi_onlinedisconnect.md "OnlineDisconnect") | Forced disconnection of a client from the server. |
| [OnlineDisconnectBatch](imtmanagerapi_onlinedisconnectbatch.md "OnlineDisconnectBatch") | Forced disconnection of multiple clients from the server. |
| [OnlineDisconnectBatchArray](imtmanagerapi_onlinedisconnectbatcharray.md "OnlineDisconnectBatchArray") | Forced disconnection of multiple clients from the server. |

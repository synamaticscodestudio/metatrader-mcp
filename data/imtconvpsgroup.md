IMTConVPSGroup
The IMTConVPS class contains the methods for getting and changing the list of groups the VPS sponsorship is allowed for:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconvpsgroup_release.md "Release") | Delete the current object. |
| [Assign](imtconvpsgroup_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconvpsgroup_clear.md "Clear") | Clear an object. |
| [Group](imtconvpsgroup_group.md "Group") | Get and set a group of accounts the VPS sponsorship is allowed for. |
| [MinBalance](imtconvpsgroup_minbalance.md "MinBalance") | Get and set the minimum balance a trader should have on the account to be able to use the sponsored VPS. |
| [InactiveDays](imtconvpsgroup_inactivedays.md "InactiveDays") | Get and set a trader inactivity period, after which the sponsored hosting is automatically canceled. |

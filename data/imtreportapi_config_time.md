Time Configuration
To get time configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [TimeCreate](imtreportapi_timecreate.md "TimeCreate") | Create an object of the time configuration. |
| [TimeCurrent](imtreportapi_timecurrent.md "TimeCurrent") | Get the current trading time. |
| [TimeGeneration](imtreportapi_timegeneration.md "TimeGeneration") | Get the time of the beginning of the current report generation. |
| [TimeGet](imtreportapi_timeget.md "TimeGet") | Get the time configuration. |

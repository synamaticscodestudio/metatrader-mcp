IMTConMessengerCountry
The IMTConMessengerCountry class contains methods for getting and setting country settings in the messenger:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconmessengercountry_release.md "Release") | Delete the current object. |
| [Assign](imtconmessengercountry_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconmessengercountry_clear.md "Clear") | Clear an object. |
| [PhoneCode](imtconmessengercountry_phonecode.md "PhoneCode") | Get and set the country calling code. |
| [MessageTemplate](imtconmessengercountry_messagetemplate.md "MessageTemplate") | Get and set a template for sending messages to users from a given country. |

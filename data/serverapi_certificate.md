Certificates
The MetaTrader 5 Server API allows managing a database of certificates that are used for extended authorization of [users](serverapi_users.md "Clients") on servers. The following functions are provided for this purpose:
| Functions | Purpose |
| --- | --- |
| --- | --- |
| [UserCertCreate](imtserverapi_usercertcreate.md "UserCertCreate") | Create an object of a certificate. |
| [UserCertUpdate](imtserverapi_usercertupdate.md "UserCertUpdate") | Add or update a client certificate. |
| [UserCertGet](imtserverapi_usercertget.md "UserCertGet") | Get the certificate of a client by the login. |
| [UserCertDelete](imtserverapi_usercertdelete.md "UserCertDelete") | Reset the user certificate. |
| [UserCertConfirm](imtserverapi_usercertconfirm.md "UserCertConfirm") | Confirm the user certificate. |

Common Configuration
Use the functions described in this section to managing the common platform configuration, as well to subscribe and unsubscribe from events associated with its change.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [CommonCreate](imtserverapi_commoncreate.md "CommonCreate") | Create a common platform configuration object. |
| [CommonCreateAllocation](imtserverapi_commoncreateallocation.md "CommonCreateAllocation") | Create an account allocation configuration object. |
| [CommonCreateAgreement](imtserverapi_commoncreateagreement.md "CommonCreateAgreement") | Create an agreement object for an account allocation configuration. |
| [CommonSubscribe](imtserverapi_commonsubscribe.md "CommonSubscribe") | Subscribe to events and hooks associated with the common configuration of the platform. |
| [CommonUnsubscribe](imtserverapi_commonunsubscribe.md "CommonUnsubscribe") | Unsubscribe from events and hooks associated with the common configuration of the platform. |
| [CommonGet](imtserverapi_commonget.md "CommonGet") | Get the common platform configuration. |
| [CommonSet](imtserverapi_commonset.md "CommonSet") | Set the common platform configuration. |

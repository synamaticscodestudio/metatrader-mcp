IMTConAutoAction
The IMTConAutoAction class contains methods for working with [actions](https://support.metaquotes.net/en/docs/mt5/platform/administration/automation/automation_action) performed upon the triggering of automation tasks:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconautoaction_release.md "Release") | Delete the current object. |
| [Assign](imtconautoaction_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconautoaction_clear.md "Clear") | Clear an object. |
| [Action](imtconautoaction_action.md "Action") | Get and set the action performed upon the triggering of an automation task. |
| [Name](imtconautoaction_name.md "Name") | Get and set the name of the action in the automation task. |
| [ParamAdd](imtconautoaction_paramadd.md "ParamAdd") | Add a parameter for an automation task action. |
| [ParamUpdate](imtconautoaction_paramupdate.md "ParamUpdate") | Change a parameter for an automation task action. |
| [ParamDelete](imtconautoaction_paramdelete.md "ParamDelete") | Delete a parameter from an automation task action. |
| [ParamClear](imtconautoaction_paramclear.md "ParamClear") | Clear the list of all parameters for an automation task action. |
| [ParamShift](imtconautoaction_paramshift.md "ParamShift") | Shift a parameter of an automation task action. |
| [ParamTotal](imtconautoaction_paramtotal.md "ParamTotal") | Get the number of parameters of an automation task action. |
| [ParamNext](imtconautoaction_paramnext.md "ParamNext") | Get the action parameter by index. |

The IMTConAutoAction class contains the following enumerations:
| Enumeration | Purpose |
| --- | --- |
| --- | --- |
| [EnActions](imtconautoaction_enum.htm#enactions) | The actions performed upon the triggering of automation tasks. |

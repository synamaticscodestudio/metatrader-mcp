Exposure Functions
Functions described in this section allow to receive information about client assets and company's coverage assets, as well as subscribe and unsubscribe from events associated with its change.
The following functions are available for working with exposure:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ExposureCreate](imtmanagerapi_exposurecreate.md "ExposureCreate") | Create an object of an exposure record. |
| [ExposureCreateArray](imtmanagerapi_exposurecreatearray.md "ExposureCreateArray") | Create an array of objects of exposure records. |
| [ExposureSubscribe](imtmanagerapi_exposuresubscribe.md "ExposureSubscribe") | Subscribe to events associated with changes in the exposure database. |
| [ExposureUnsubscribe](imtmanagerapi_exposureunsubscribe.md "ExposureUnsubscribe") | Undubscribe from events associated with changes in the exposure database. |
| [ExposureCurrency](imtmanagerapi_exposurecurrency.md "ExposureCurrency") | Get and set a currency of the net total of assets. |
| [ExposureTotal](imtmanagerapi_exposuretotal.md "ExposureTotal") | Get the number of records in the exposure table. |
| [ExposureNext](imtmanagerapi_exposurenext.md "ExposureNext") | Get a record from an exposure table by an index. |
| [ExposureGet](imtmanagerapi_exposureget.md "ExposureGet") | Get a record from an exposure table by a symbol. |
| [ExposureGetAll](imtmanagerapi_exposuregetall.md "ExposureGetAll") | Get an array of exposure records. |

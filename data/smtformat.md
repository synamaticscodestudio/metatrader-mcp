SMTFormat
This class is used for formatting strings. It contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [FormatError](smtformat_formaterror.md "FormatError") | Converts an error code used in the MetaTrader 5 platform into a text description. |
| [FormatDouble](smtformat_formatdouble.md "FormatDouble") | Formats a number with a given number of decimal places to a string. |
| [FormatMoney](smtformat_formatmoney.md "FormatMoney") | Formats a number with a given number of decimal places to a string. It is used to show sums of money with thousands separated. |
| [FormatPrice](smtformat_formatprice.md "FormatPrice") | Formats a number into a string. Basic and additional accuracy are separately set for a number. As a rule, is used to format the position price. |
| [FormatPrices](smtformat_formatprices.md "FormatPrices") | Formats values of Bid, Ask and Last from a passed structure MTTickShort to a string. |
| [FormatVolume](smtformat_formatvolume.md "FormatVolume") | Formats volume to a string. |
| [FormatVolumeDouble](smtformat_formatvolumedouble.md "FormatVolumeDouble") | Formats order volume (double) to a string. |
| [FormatVolumeExt](smtformat_formatvolumeext.md "FormatVolumeExt") | Formats extended accuracy volume to a string. |
| [FormatSize](smtformat_formatsize.md "FormatSize") | Formats size in units to a string. |
| [FormatVolumeOrder](smtformat_formatvolumeorder.md "FormatVolumeOrder") | Formats order volume to a string. |
| [FormatVolumeExtOrder](smtformat_formatvolumeextorder.md "FormatVolumeExtOrder") | Formats order volume specified with extended accuracy to a string. |
| [FormatSizeOrder](smtformat_formatsizeorder.md "FormatSizeOrder") | Formats order size in units to a string. |
| [FormatDateTime](smtformat_formatdatetime.md "FormatDateTime") | Formats date and time to a string. |
| [FormatDateTimeMsc](smtformat_formatdatetimemsc.md "FormatDateTimeMsc") | Formats date and time into a string with an indication of milliseconds. |
| [FormatTime](smtformat_formattime.md "FormatTime") | Formats time into a string. |
| [FormatTimeMsc](smtformat_formattimemsc.md "FormatTimeMsc") | Formats time into a string with an indication of milliseconds. |
| [FormatIP](smtformat_formatip.md "FormatIP") | Formats an IP address and a port to a string. |
| [FormatPositionType](smtformat_formatpositiontype.md "FormatPositionType") | Formats the type of a position in a string with a text description. |
| [FormatOrderType](smtformat_formatordertype.md "FormatOrderType") | Formats the type of an order in a string with a text description. |
| [FormatOrderStatus](smtformat_formatorderstatus.md "FormatOrderStatus") | Formats the state of an order in a string with a text description. |
| [FormatOrderTypeFilling](smtformat_formatordertypefilling.md "FormatOrderTypeFilling") | Formats order filling type in a string with a text description. |
| [FormatOrderTypeTime](smtformat_formatordertypetime.md "FormatOrderTypeTime") | Formats order expiration type in a string with a text description. |
| [FormatOrderTypeReason](smtformat_formatordertypereason.md "FormatOrderTypeReason") | Formats the order placing reason in a string with a text description. |
| [FormatOrderPrice](smtformat_formatorderprice.md "FormatOrderPrice") | Formats the order price and its triggering price (if any) in a string with a text description. |
| [FormatDealAction](smtformat_formatdealaction.md "FormatDealAction") | Formats the type of action performed by a deal to a string with a text description. |
| [FormatDealEntry](smtformat_formatdealentry.md "FormatDealEntry") | Formats the type of action performed by a deal relative to a position, to a string with a text description. |

News Database Functions
Using the functions, you can create and send news, as well as subscribe and unsubscribe from events with changes in the news database.
The following functions are available for working with news:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [NewsCreate](imtadminapi_newscreate.md "NewsCreate") | Create an object of a news item. |
| [NewsSubscribe](imtadminapi_newssubscribe.md "NewsSubscribe") | Subscribe to events and hooks associated with changes in the news database. |
| [NewsUnsubscribe](imtadminapi_newsunsubscribe.md "NewsUnsubscribe") | Undubscribe from events and hooks associated with changes in the news database. |
| [NewsSend](imtadminapi_newssend.md "NewsSend") | Send news. |

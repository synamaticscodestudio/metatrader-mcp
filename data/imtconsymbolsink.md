IMTConSymbolSink
The IMTConSymbolSink class contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnSymbolAdd](imtconsymbolsink_onsymboladd.md "OnSymbolAdd") | A handler of the event of adding a new symbol. |
| [OnSymbolUpdate](imtconsymbolsink_onsymbolupdate.md "OnSymbolUpdate") | A handler of the event of updating symbol settings. |
| [OnSymbolDelete](imtconsymbolsink_onsymboldelete.md "OnSymbolDelete") | A handler of the event of symbol removal. |
| [OnSymbolSync](imtconsymbolsink_onsymbolsync.md "OnSymbolSync") | A handler of the event of symbols synchronization. |
| [HookSymbolAdd](imtconsymbolsink_hooksymboladd.md "HookSymbolAdd") | Hook for adding of the new symbol. Only used in Server API. |
| [HookSymbolUpdate](imtconsymbolsink_hooksymbolupdate.md "HookSymbolUpdate") | Hook for the update of symbol settings. Only used in Server API. |
| [HookSymbolDelete](imtconsymbolsink_hooksymboldelete.md "HookSymbolDelete") | Symbol deletion hook. Only used in Server API. |

Plugins
The following requests are provided for operations with plugin settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/plugin/add](webapi_plugin_add.md "Add") | Add and update a plugin configuration in the trading platform. |
| [/api/plugin/delete](webapi_plugin_delete.md "Delete") | Delete a plugin configuration from the trading platform. |
| [/api/plugin/shift](webapi_plugin_shift.md "Shift") | Change the position of a plugun configuration in the list. |
| [/api/plugin/total](webapi_plugin_total.md "Get Total") | Get the total number of plugin configurations existing in the platform. |
| [/api/plugin/next](webapi_plugin_next.md "Get by Index") | Get the configuration of one or more plugins by index in the list. |
| [/api/plugin/get](webapi_plugin_get.md "Get by Name") | Get plugin configuration by its name. |
| [/api/plugin/get_all](webapi_plugin_get_all.md "Get All") | Get all existing plugin configurations. |
| [/api/plugin/module/total](webapi_plugin_module_total.md "Get Total Modules") | Get the number of plugin modules available in the platform. |
| [/api/plugin/module/next](webapi_plugin_module_next.md "Get Module by Index") | Get the description of a plugin module by the index in the list. |
| [/api/plugin/module/get](webapi_plugin_module_get.md "Get Module by Name") | Get the description of a plugin module by its name. |

Holiday Configuration
Functions allow managing the holidays configuration, as well subscribe and unsubscribe from events associated with its change.
The following functions for configuring holidays are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [HolidayCreate](imtadminapi_holidaycreate.md "HolidayCreate") | Create an object of the configuration of holidays. |
| [HolidaySubscribe](imtadminapi_holidaysubscribe.md "HolidaySubscribe") | Subscribe to events associated with the configuration of holidays. |
| [HolidayUnsubscribe](imtadminapi_holidayunsubscribe.md "HolidayUnsubscribe") | Unsubscribe from events associated with the configuration of holidays. |
| [HolidayUpdate](imtadminapi_holidayupdate.md "HolidayUpdate") | Add or update a holiday configuration. |
| [HolidayUpdateBatch](imtadminapi_holidayupdatebatch.md "HolidayUpdateBatch") | Add or edit multiple holiday configurations. |
| [HolidayDelete](imtadminapi_holidaydelete.md "HolidayDelete") | Delete a holiday configuration by the index. |
| [HolidayDeleteBatch](imtadminapi_holidaydeletebatch.md "HolidayDeleteBatch") | Delete multiple holiday configurations. |
| [HolidayShift](imtadminapi_holidayshift.md "HolidayShift") | Changes the position of a holiday configuration in the list. |
| [HolidayTotal](imtadminapi_holidaytotal.md "HolidayTotal") | The total number of configurations of holidays available in the platform. |
| [HolidayNext](imtadminapi_holidaynext.md "HolidayNext") | Gets a holiday configuration with the specified index. |

IMTReportCacheValue
IMTReportCacheValue is the [data cache](imtreportapi_cache.md "Data cache") key description interface. It is used for working with the cache values and for iteration between keys.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtreportcachevalue_release.md "Release") | Delete the current object. |
| [Key](imtreportcachevalue_key.md "Key") | Get a key. |
| [Data](imtreportcachevalue_data.md "Data") | Get the key value. |
| [Size](imtreportcachevalue_size.md "Size") | Get the value of the key size type. |
| [Next](imtreportcachevalue_next.md "Next") | Go to the next key. |

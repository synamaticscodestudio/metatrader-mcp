Groups
Use the methods described in this section to obtain the parameters of client groups on the server:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [GroupCreate](php_groupcreate.md "GroupCreate") | Create an object of a client group. |
| [GroupAdd](php_groupadd.md "GroupAdd") | Add or change a group configuration on the server. |
| [GroupDelete](php_groupdelete.md "GroupDelete") | Delete a group configuration with the specified name. |
| [GroupTotal](php_grouptotal.md "GroupTotal") | Get the number of groups created on the trade server. |
| [GroupNext](php_groupnext.md "GroupNext") | Get the group configuration by its index in the list of groups of the platform. |
| [GroupGet](php_groupget.md "GroupGet") | Get the group configuration by its name. |

The IMTConEmail class contains methods for getting and changing mail server configurations:

| --- | --- |
| --- | --- |

| [Assign](imtconemail_assign.md "Assign") | Assign a passed object to the current one. |

| [SenderMail](imtconemail_sendermail.md "SenderMail") | Get and set the email address, from which emails will be sent via the mail server configurations. |

| [Server](imtconemail_server.md "Server") | Get and set the SMTP server address in the mail server configuration. |
| [Login](imtconemail_login.md "Login") | Get and set the SMTP login in the mail server configuration. |
| [Password](imtconemail_password.md "Password") | Get and set the SMTP password in the mail server configuration. |

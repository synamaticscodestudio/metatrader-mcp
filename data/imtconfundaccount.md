IMTConFundAccount
The IMTConFundAccount class contains methods for getting and changing the list of [fund managers](https://support.metaquotes.net/en/docs/mt5/platform/administration/fund_etf#trade_accounts):
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconfundaccount_release.md "Release") | Delete the current object. |
| [Assign](imtconfundaccount_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconfundaccount_clear.md "Clear") | Clear an object. |
| [Login](imtconfundaccount_login.md "Login") | Get and set the login of the account used by the manager for fund management operations. |
| [Name](imtconfundaccount_name.md "Name") | Get the name of the fund manager. |
| [Balance](imtconfundaccount_balance.md "Balance") | Get the current balance of the account used for fund management. |
| [Equity](imtconfundaccount_equity.md "Equity") | Get the current equity of the account used for fund management. |
| [Currency](imtconfundaccount_currency.md "Currency") | Get the currency in which manager account balance and equity are specified. |
| [CurrencyDigits](imtconfundaccount_currencydigits.md "CurrencyDigits") | Get the number of decimal places in the manager account balance and equity values. |

Configuration of Groups
Functions allow to manage groups, as well subscribe and unsubscribe from events associated with their change.
The following functions for managing groups are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [GroupCreate](imtadminapi_groupcreate.md "GroupCreate") | Create an object of the group configuration. |
| [GroupSymbolCreate](imtadminapi_groupsymbolcreate.md "GroupSymbolCreate") | Create an object of symbol configuration for a group. |
| [GroupCommissionCreate](imtadminapi_groupcommissioncreate.md "GroupCommissionCreate") | Create an object of commission configuration for a group. |
| [GroupTierCreate](imtadminapi_grouptiercreate.md "GroupTierCreate") | Create an object of commission range configuration for a group. |
| [GroupSubscribe](imtadminapi_groupsubscribe.md "GroupSubscribe") | Subscribe to events associated with the configuration of groups. |
| [GroupUnsubscribe](imtadminapi_groupunsubscribe.md "GroupUnsubscribe") | Unsubscribe from events associated with the configuration of groups. |
| [GroupUpdate](imtadminapi_groupupdate.md "GroupUpdate") | Adds or updates a group configuration. |
| [GroupUpdateBatch](imtadminapi_groupupdatebatch.md "GroupUpdateBatch") | Add or edit multiple group configurations. |
| [GroupDelete](imtadminapi_groupdelete.md "GroupDelete") | Delete a group configuration by the name or index |
| [GroupDeleteBatch](imtadminapi_groupdeletebatch.md "GroupDeleteBatch") | Delete multiple group configurations. |
| [GroupShift](imtadminapi_groupshift.md "GroupShift") | Changes the position of a group configuration in the list. |
| [GroupTotal](imtadminapi_grouptotal.md "GroupTotal") | The total number of group configurations available in the platform. |
| [GroupNext](imtadminapi_groupnext.md "GroupNext") | Get the group configuration by the index. |
| [GroupGet](imtadminapi_groupget.md "GroupGet") | Get the group configuration by the name. |

Price Data
The MetaTrader 5 API provides interfaces for working with the platform's price data. The interfaces can be used to add quotes to the price stream, track incoming quotes and update the quotes if necessary.
The following interfaces are provided for working with price data:
  * [IMTTickSink](imtticksink.md "IMTTickSink") — tick data events and hooks.
  * [IMTChartSink](imtchartsink.md "IMTChartSink") — minute data events and hooks.

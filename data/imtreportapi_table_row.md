Rows Functions
The following functions for managing rows are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [TableRowClear](imtreportapi_tablerowclear.md "TableRowClear") | Delete the contents of a whole table. |
| [TableRowWrite](imtreportapi_tablerowwrite.md "TableRowWrite") | Add (output) one record to a table. |
| [TableRowTotal](imtreportapi_tablerowtotal.md "TableRowTotal") | Get the number of rows in a table. |

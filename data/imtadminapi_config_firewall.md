Firewall Configuration
Functions allow managing the firewall configuration, as well subscribe and unsubscribe from events associated with its change.
To configure a firewall provides the configuring the firewall:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [FirewallCreate](imtadminapi_firewallcreate.md "FirewallCreate") | Create an object of the firewall configuration. |
| [FirewallSubscribe](imtadminapi_firewallsubscribe.md "FirewallSubscribe") | Subscribe to events associated with the firewall configuration. |
| [FirewallUnsubscribe](imtadminapi_firewallunsubscribe.md "FirewallUnsubscribe") | Unsubscribe from events associated with the firewall configuration. |
| [FirewallUpdate](imtadminapi_firewallupdate.md "FirewallUpdate") | Add or update a firewall configuration. |
| [FirewallUpdateBatch](imtadminapi_firewallupdatebatch.md "FirewallUpdateBatch") | Add or edit multiple firewall configurations. |
| [FirewallDelete](imtadminapi_firewalldelete.md "FirewallDelete") | Delete a configuration of a data feed by its index. |
| [FirewallDeleteBatch](imtadminapi_firewalldeletebatch.md "FirewallDeleteBatch") | Delete multiple firewall configurations. |
| [FirewallShift](imtadminapi_firewallshift.md "FirewallShift") | Change the position of a firewall configuration in the list. |
| [FirewallTotal](imtadminapi_firewalltotal.md "FirewallTotal") | The total number of firewall configurations available in the platform. |
| [FirewallNex](imtadminapi_firewallnext.md "FirewallNext") | Get the firewall configuration by the index. |

Operations with Connection
This section describes methods for working with Manager API network connection. The following methods are available:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [NetworkRescan](imtmanagerapi_networkrescan.md "NetworkRescan") | Forced launch of scanning for access points and receiving the scanning results. |
| [NetworkBytesSent](imtmanagerapi_networkbytessent.md "NetworkBytesSent") | Receive the number of bytes sent to the server via Manager API. |
| [NetworkBytesRead](imtmanagerapi_networkbytesread.md "NetworkBytesRead") | Receive the number of bytes accepted from the server via Manager API. |
| [NetworkServer](imtmanagerapi_networkserver.md "NetworkServer") | Receive the name of the server Manager API is currently connected to. |
| [NetworkAddress](imtmanagerapi_networkaddress.md "NetworkAddress") | Receive IP address of the server Manager API is currently connected to. |

Spreads
The following requests are provided for retrieving platform spread settings:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/spread/add](webapi_spread_add.md "Add") | Add and update spread configurations in the trading platform. |
| [/api/spread/delete](webapi_spread_delete.md "Delete") | Delete spread configurations from the trading platform. |
| [/api/spread/shift](webapi_spread_shift.md "Shift") | Change the position of a spread configuration in the list. |
| [/api/spread/total](webapi_spread_total.md "Get Total") | Get the total number of spread configurations existing in the platform. |
| [/api/spread/next](webapi_spread_next.md "Get by Index") | Get one or more spread configurations by index in the list. |
| [/api/spread/get_all](webapi_spread_get_all.md "Get All") | Get all exiting spread configurations. |

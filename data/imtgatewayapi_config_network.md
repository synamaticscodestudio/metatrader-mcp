Network Configuration
Functions allow managing the configuration of the platform components, as well subscribe and unsubscribe from events associated with its change.
To manage the configuration of components, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [NetServerCreate](imtgatewayapi_netservercreate.md "NetServerCreate") | Create an object of the network configuration. |
| [NetServerRangeCreate](imtgatewayapi_netserverrangecreate.md "NetServerRangeCreate") | Create an object of the range of orders, deals or accounts. |
| [NetServerSubscribe](imtgatewayapi_netserversubscribe.md "NetServerSubscribe") | Subscribe to events associated with the network configuration. |
| [NetServerUnsubscribe](imtgatewayapi_netserverunsubscribe.md "NetServerUnsubscribe") | Unsubscribe from events associated with the network configuration. |
| [NetServerTotal](imtgatewayapi_netservertotal.md "NetServerTotal") | The total number of server configurations available in the platform. |
| [NetServerNext](imtgatewayapi_netservernext.md "NetServerNext") | Get a server configuration by the index. |
| [NetServerGet](imtgatewayapi_netserverget.md "NetServerGet") | Get a server configuration by the ID. |

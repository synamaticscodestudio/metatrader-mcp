Custom Functions
MetaTrader 5 Manager API provides the possibility of extending the protocol for running custom commands on the server. Currently, there are two functions for this purpose:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [CustomCommand](imtmanagerapi_customcommand.md "CustomCommand") | Sends a custom command to the server. |
| [CustomCreateStream](imtmanagerapi_customcreatestream.md "CustomCreateStream") | Creation of an object of a byte stream. |

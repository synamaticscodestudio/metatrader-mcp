IMTConEmailSink
The IMTConEmailSink class contains event handlers for changes in mail server configurations:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnEmailAdd](imtconemailsink_onemailadd.md "OnEmailAdd") | Handler of the addition of a new mail server configuration. |
| [OnEmailUpdate](imtconemailsink_onemaildelete.md "OnEmailDelete") | Handler of the update of a mail server configuration. |
| [OnEmailDelete](imtconemailsink_onemaildelete.md "OnEmailDelete") | Handler of the deletion of a mail server configuration. |
| [OnEmailSync](imtconemailsink_onemailsync.md "OnEmailSync") | Handler of the synchronization of mail server configurations. |

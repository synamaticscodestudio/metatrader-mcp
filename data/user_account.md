Accounts
MetaTrader 5 API provides access to users' trading accounts. Two interfaces are available for working with trading accounts:
  * [IMTAccount](imtaccount.md "IMTAccount") Provides the possibility to receive information about the current account state.
  * [IMTAccountArray](imtaccountarray.md "IMTAccountArray") An interface for working with the arrays of trade accounts.
  * [IMTAccountSink](imtaccountsink.md "IMTAccountSink") An interface for working with the events related to changes in the account trading state.

User Functions
The functions allow receiving the data on users and their trade accounts from a trade server database. The following functions are available for working with users:
| Functions | Purpose |
| --- | --- |
| --- | --- |
| [UserCreate](imtreportapi_usercreate.md "UserCreate") | Create an object of a client record. |
| [UserCreateAccount](imtreportapi_usercreateaccount.md "UserCreateAccount") | Create an object of a client's trading account. |
| [UserGet](imtreportapi_userget.md "UserGet") | Get a client record by the login. |
| [UserGetLight](imtreportapi_usergetlight.md "UserGetLight") | Get the client record brief description. |
| [UserLogins](imtreportapi_userlogins.md "UserLogins") | Returns an array of logins of the clients who are included in the specified group. |
| [UserAccountGet](imtreportapi_useraccountget.md "UserAccountGet") | Get client trading account by a login. |
| [UserSelect](imtreportapi_userselect.md "UserSelect") | Request accounts from a database according to specified criteria. |
| [UserAccountSelect](imtreportapi_useraccountselect.md "UserAccountSelect") | Request trading account from a database according to specified criteria. |
| [ClientSelect](imtreportapi_clientselect.md "ClientSelect") | Request clients from a database according to specified criteria. |

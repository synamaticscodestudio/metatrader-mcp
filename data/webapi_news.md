News
The MetaTrader 5 Web API allows receiving and sending news to terminals via the platform's internal news system. The following requests are provided:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/news/send](webapi_news_send.md "Send News") | Send news. |
| [/api/news/get](webapi_news_get.md "Get News Without Body") | Get a brief news description without a body. |
| [/api/news/get_body](webapi_news_body_request.md "Get News With Body") | Get the full news description with its body. |

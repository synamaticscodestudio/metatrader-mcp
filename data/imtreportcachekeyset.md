IMTReportCacheKeySet
IMTReportCacheKeySet is an auxiliary interface for describing a set of [data cache](imtreportapi_cache.md "Data cache") keys. Use the interface to receive cache information for multiple keys without having to perform routine operations related to request preparation.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtreportcachekeyset_release.md "Release") | Delete the current object. |
| [Assign](imtreportcachekeyset_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtreportcachekeyset_clear.md "Clear") | Clear an object. |
| [Swap](imtreportcachekeyset_swap.md "Swap") | Exchange key sets with the passed object. |
| [Total](imtreportcachekeyset_total.md "Total") | Get the number of keys in the set. |
| [Array](imtreportcachekeyset_array.md "Array") | Get keys from the set as a sorted array. |
| [Next](imtreportcachekeyset_next.md "Next") | Get the next key from a sorted key array. |
| [Search](imtreportcachekeyset_search.md "Search") | Search key in the array. |
| [ContainsSet](imtreportcachekeyset_containsset.md "ContainsSet") | Check if the key set contains all the keys from the passed set. |
| [Reserve](imtreportcachekeyset_reserve.md "Reserve") | Reserve memory for keys in a set. |
| [Insert](imtreportcachekeyset_insert.md "Insert") | Add a key to the current set. |
| [InsertArray](imtreportcachekeyset_insertarray.md "InsertArray") | Add an array of keys to the current set. |

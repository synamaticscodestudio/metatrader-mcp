Messengers
The following requests are provided for operations with messengers settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/messenger/add](webapi_messenger_add.md "Add") | Add and update messenger configurations in the trading platform. |
| [/api/messenger/delete](webapi_messenger_delete.md "Delete") | Delete messenger configurations from the trading platform. |
| [/api/messenger/shift](webapi_messenger_shift.md "Shift") | Change the position of a messenger configuration in the list. |
| [/api/messenger/total](webapi_messenger_total.md "Get Total") | Get the total number of messenger configurations available in the platform. |
| [/api/messenger/next](webapi_messenger_next.md "Get by Index") | Get a messenger configuration by the index in the list. |
| [/api/messenger/get](webapi_messenger_get.md "Get by Name") | Get a messenger configuration by name. |
| [/api/messenger/get_all](webapi_messenger_get_all.md "Get All") | Get all existing messenger configurations. |
| [/api/messenger/send](webapi_messenger_send.md "Send Message") | Send an SMS message. |

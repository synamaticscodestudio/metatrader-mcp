Daily Reports Database Functions
The functions described in this section allow to observe the daily reports generated by the MetaTrader 5 trade server. The following functions are available for working with a daily reports base:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DailyCreate](imtreportapi_dailycreate.md "DailyCreate") | Create an object of a daily report. |
| [DailyCreateArray](imtreportapi_dailycreatearray.md "DailyCreateArray") | Create an object of the array of daily reports. |
| [DailyGet](imtreportapi_dailyget.md "DailyGet") | Get daily reports. |
| [DailyGetLight](imtreportapi_dailygetlight.md "DailyGetLight") | Get eased daily reports. |
| [DailySelect](imtreportapi_dailyselect.md "DailySelect") | Request daily reports from a database according to specified criteria. |

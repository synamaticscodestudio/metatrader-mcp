Reports
The following requests are provided for operations with report settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/report/add](webapi_report_add.md "Add") | Add and update a report configuration in the trading platform. |
| [/api/report/delete](webapi_report_delete.md "Delete") | Delete a report configuration from the trading platform. |
| [/api/report/shift](webapi_report_shift.md "Shift") | Change the position of a report configuration in the list. |
| [/api/report/total](webapi_report_total.md "Get Total") | Get the total number of report configurations existing in the platform. |
| [/api/report/next](webapi_report_next.md "Get by Index") | Get the configuration of one or more reports by index in the list. |
| [/api/report/get](webapi_report_get.md "Get by Name") | Get a report configuration by its name. |
| [/api/report/get_all](webapi_report_get_all.md "Get All") | Get all existing report configurations. |
| [/api/report/module/total](webapi_report_module_total.md "Get Total Modules") | Get the number of report modules available in the platform. |
| [/api/report/module/next](webapi_report_module_next.md "Get Module by Index") | Get the description of a report module by the index in the list. |
| [/api/report/module/get](webapi_report_module_get.md "Get Module by Name") | Get the description of a report module by its name. |

IMTAttachment
The IMTAttachment class is designed for working with [document](imtdocument.md "IMTDocument") files and [attachments in comments](imtcomment.md "IMTComment"). The interface contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtattachment_release.md "Release") | Delete the current object. |
| [Assign](imtattachment_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtattachment_clear.md "Clear") | Clear an object. |
| [RecordID](imtattachment_recordid.md "RecordID") | Get and set the attachment identifier. |
| [RelatedClient](imtattachment_relatedclient.md "RelatedClient") | Get and set the client ID with which the attachment is associated. |
| [FileType](imtattachment_filetype.md "FileType") | Get and set the file type. |
| [FileName](imtattachment_filename.md "FileName") | Get and set the file name. |
| [FileContent](imtattachment_filecontent.md "FileContent") | Get and set the attached file contents. |
| [FileSize](imtattachment_filesize.md "FileSize") | Get and set the attached file size. |
| [FileFlags](imtattachment_fileflags.md "FileFlags") | Get and set file flags. |

The IMTAttachment class contains the following enumerations:
| Enumeration | Purpose |
| --- | --- |
| --- | --- |
| [EnFileType](imtattachment_enum.htm#enfiletype) | File types. |
| [EnFileFlags](imtattachment_enum.htm#enfileflags) | File flags. |

Structures
The MetaTrader 5 API provides several pre-defined structures that are designed for storing and passing of service information: Virtually all APIs use the same structures.
  * [MTProxyInfo](mtproxyinfo.md "MTProxyInfo") — parameters of connection through a proxy server.
  * [MTLicenseCheck](mtlicensecheck.md "MTLicenseCheck") — Information for verifying the license.
  * [MTTick](mttick.md "MTTick") — a full description of a tick.
  * [MTTickShort](mttickshort.md "MTTickShort") — a brief description of a tick.
  * [MTTickRate](mttickrate.md "MTTickRate") — a brief description of a tick.
  * [MTTickStat](mttickstat.md "MTTickStat") — description of statistical information about ticks.
  * [MTMailRange](mtmailrange.md "MTMailRange") — a range of recipients of a mailing list.
  * [MTLogRecord](mtlogrecord.md "MTLogRecord") — description of a log entry.
  * [MTChartBar](mtchartbar.md "MTChartBar") — description of a chart bar.
  * [MTBookItem](mtbookitem.md "MTBookItem") — description of an element of the Depth of Market.
  * [MTBook/MTBookDiff](mtbook.md "MTBook/MTBookDiff") — description of the Depth of Market.
  * [MTGatewayInfo](mtgatewayinfo.md "MTGatewayInfo") — gateway/data feed module parameters.
  * [MTNews](mtnews.md "MTNews") — news description.
  * [MTEconomicEvent](mteconomicevent.md "MTEconomicEvent") — description of a news of the economic calendar.
  * [MTReportInfo](mtreportinfo.md "MTReportInfo") — initial information about the report module.
  * [MTReportParam](mtreportparam.md "MTReportParam") — module parameters.
  * [MTReportServerInfo](mtreportserverinfo.md "MTReportServerInfo") — information about the trading platform and the server.
  * [MTPluginInfo](mtplugininfo.md "MTPluginInfo") — initial information about the plugin.
  * [MTPluginParam](mtpluginparam.md "MTPluginParam") — default parameters of the plugin.
  * [MTServerInfo](mtserverinfo.md "MTServerInfo") — information about the trading platform and the server.

Email
The following requests are provided for operations with mail server settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/email/add](webapi_email_add.md "Add") | Add and update mail server configurations in the trading platform. |
| [/api/email/delete](webapi_email_delete.md "Delete") | Delete mail server configurations from the trading platform. |
| [/api/email/shift](webapi_email_shift.md "Shift") | Change the position of a mail server configuration in the list. |
| [/api/email/total](webapi_email_total.md "Get Total") | Get the total number of mail server configurations available in the platform. |
| [/api/email/next](webapi_email_next.md "Get by Index") | Get the configuration of one or more mail servers by index in the list. |
| [/api/email/get](webapi_email_get.md "Get by Name") | Get a mail server configuration by name. |
| [/api/email/get_all](webapi_email_get_all.md "Get All") | Get all existing mail server configurations. |
| [/api/email/send](webapi_email_send.md "Send Email") | Send an email to a selected address. |

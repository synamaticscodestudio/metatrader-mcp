Byte Stream
MetaTrader 5 API includes the special interface [IMTByteStream](imtbytestream.md "IMTByteStream"), which provides for an easier operation with unformatted byte data streams. These data can be passed using [user functions](imtmanagerapi_custom.md "Custom Functions").

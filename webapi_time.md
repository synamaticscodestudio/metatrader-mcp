Timing
The following requests are provided for operations with working hours settings in the trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/time/server](webapi_time_server.md "Get Server Time") | Get the current server time. |
| [/api/time/get](webapi_time_get.md "Get Settings") | Get the working time settings of the server. |
| [/api/time/set](webapi_time_set.md "Update Settings") | Update the working time settings of the server. |

The format, in which the data about time configuration is passed, is described under the ["Data Structure"](webapi_time_data_structure.md "Data Structure") section.

Configuration of Additional Parameters
The additional parameter configuration interfaces are auxiliary. They provide access to parameters of the "Name-Value" type. For example, such parameters are used for [configuring gateways](config_gateway.md "Gateways"):
Shown above:
  * [Parameter name](imtconparam_name.md "Name").
  * [Parameter value](imtconparam_value.md "Value").

The following interfaces are available for configuring "Name-Value" parameters:
  * [IMTConParam](imtconparam.md "IMTConParam") The interface allows managing the parameter object, allowing access to its name, type and value.
  * [IMTConParamArray](imtconparamarray.md "IMTConParamArray") The interface allows managing the object of the array of parameters.

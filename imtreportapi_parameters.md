Report Parameters
To manage generated reports parameters, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ParamCreate](imtreportapi_paramcreate.md "ParamCreate") | Create an object of a report parameter. |
| [ParamTotal](imtreportapi_paramtotal.md "ParamTotal") | Get the number of parameters set by a manager during report generation. |
| [ParamGet](imtreportapi_paramget.md "ParamGet") | Get a report parameter set in a manager terminal by its name. |
| [ParamNext](imtreportapi_paramnext.md "ParamNext") | Get a report parameter set from a manager terminal by its position. |
| [ParamLogins](imtreportapi_paramlogins.md "ParamLogins") | Get an array of the logins, for which a report from a manager terminal is requested. |
| [ParamFrom](imtreportapi_paramfrom.md "ParamFrom") | Get the beginning of the time range set in a manager terminal, for which a report is generated. |
| [ParamTo](imtreportapi_paramto.md "ParamTo") | Get the end of the time range set in a manager terminal, for which a report is generated. |
| [ParamGroups](imtreportapi_paramgroups.md "ParamGroups") | Get the "Groups" parameter value line set in a manager terminal. |
| [ParamSymbols](imtreportapi_paramsymbols.md "ParamSymbols") | Get the "Symbols" parameter value line set in a manager terminal. |
| [ParamIEVersion](imtreportapi_paramieversion.md "ParamIEVersion") | Get the Internet Explorer version installed on the computer with the manager terminal requesting a report. |
| [ParamScreenWidth](imtreportapi_paramscreenwidth.md "ParamScreenWidth") | Get a screen resolution by width on the computer with the manager terminal requesting a report. |
| [ParamScreenHeight](imtreportapi_paramscreenheight.md "ParamScreenHeight") | Get a screen resolution by height on the computer with the manager terminal requesting a report. |

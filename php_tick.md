Prices
Use the methods described in this section to get price data from a trade server. The following methods are available:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [TickLast](php_ticklast.md "TickLast") | Get the current prices of symbols. |
| [TickLastGroup](php_ticklastgroup.md "TickLastGroup") | Get the current prices of symbols taking into account conversion for the specified group. |
| [TickStat](php_tickstat.md "TickStat") | Get statistical data of symbols. |

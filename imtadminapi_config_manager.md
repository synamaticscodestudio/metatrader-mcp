Configuration of Managers
Functions allow managing the configurations of managers, as well subscribe and unsubscribe from events associated with their change.
The following functions for configuring mangers are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ManagerCreate](imtadminapi_managercreate.md "ManagerCreate") | Create a manager configuration object. |
| [ManagerAccessCreate](imtadminapi_manageraccesscreate.md "ManagerAccessCreate") | Create an object of configuration of an access list by IP addresses for a manager. |
| [ManagerReportCreate](imtadminapi_managerreportcreate.md "ManagerReportCreate") | Create an object for the manager's permission to access the report. |
| [ManagerSubscribe](imtadminapi_managersubscribe.md "ManagerSubscribe") | Subscribe to events associated with the configuration of managers. |
| [ManagerUnsubscribe](imtadminapi_managerunsubscribe.md "ManagerUnsubscribe") | Unsubscribe from events associated with the configuration of managers. |
| [ManagerUpdate](imtadminapi_managerupdate.md "ManagerUpdate") | Add or update a manager configuration. |
| [ManagerUpdateBatch](imtadminapi_managerupdatebatch.md "ManagerUpdateBatch") | Add or edit multiple manager configurations. |
| [ManagerDelete](imtadminapi_managerdelete.md "ManagerDelete") | Deletes a manager configuration by the index. |
| [ManagerDeleteBatch](imtadminapi_managerdeletebatch.md "ManagerDeleteBatch") | Delete multiple manager configurations. |
| [ManagerShift](imtadminapi_managershift.md "ManagerShift") | Change the position of a manager configuration in the list. |
| [ManagerTotal](imtadminapi_managertotal.md "ManagerTotal") | The total number of manager configurations available in the platform. |
| [ManagerNext](imtadminapi_managernext.md "ManagerNext") | Get a manager configuration with the specified index. |
| [ManagerGet](imtadminapi_managerget.md "ManagerGet") | Get a manager configuration with the specified login. |
| [ManagerCurrent](imtadminapi_managercurrent.md "ManagerCurrent") | Get the configuration of the current manager account. |

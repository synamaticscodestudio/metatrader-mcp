Protocol Implementation in PHP
The MetaTrader 5 SDK includes an example of [implementation of the Web API protocol](mtwebapi.md "MTWebAPI Class") in the PHP language. To show how to use this implementation, the package includes two examples: a [ready-made project](php_examples.md "Examples") of registration of accounts in the trading platform through a web page, and an example of implementation of custom commands. The protocol implementation and the examples are located in /Examples/PHP folder of the Web API installation directory. The examples directory contains the following folders and files:
| Folder | Files |
| --- | --- |
| --- | --- |
| mt5_api | Implementation of the MetaTrader 5 Web API protocol in the PHP language. |
| web_registration | Example of Web registration of accounts in the MetaTrader 5 platform. |
| api_extension | Example of implementation of custom commands. |

Orders
The functions allow to observe a server trading orders base. The following functions are available for this purpose:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [OrderCreate](imtreportapi_ordercreate.md "OrderCreate") | Create an object of a trade order. |
| [OrderCreateArray](imtreportapi_ordercreatearray.md "OrderCreateArray") | Create an object of the array of orders. |
| [OrderGet](imtreportapi_orderget.md "OrderGet") | Gets open orders of a client. |
| [OrderSelect](imtreportapi_orderselect.md "OrderSelect") | Request open orders from a database according to specified criteria. |
| [HistoryGet](imtreportapi_historyget.md "HistoryGet") | Get closed orders of a client. |
| [HistorySelect](imtreportapi_historyselect.md "HistorySelect") | Request closed orders from a database according to specified criteria. |

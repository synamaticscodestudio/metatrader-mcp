Client Connection
The functions described in this section provide control of connections to gateways/data feeds using logins, passwords and IP-addresses.
The following functions are provided:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ClientAdd](imtgatewayapi_clientadd.md "ClientAdd") | Adding permission for the platform components connection using a specified login and a password. |
| [ClientAllowIP](imtgatewayapi_clientallowip.md "ClientAllowIP") | Adding permission for the connection from a specified IP address. |

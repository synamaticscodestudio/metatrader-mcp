Report Configuration
Functions allow managing configurations of reports, as well subscribe and unsubscribe from events associated with their change.
The following functions for managing reports are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ReportCreate](imtadminapi_reportcreate.md "ReportCreate") | Create an object of the configuration of reports. |
| [ReportModuleCreate](imtadminapi_reportmodulecreate.md "ReportModuleCreate") | Create an object of configuration of a report module. |
| [ReportParamCreate](imtadminapi_reportparamcreate.md "ReportParamCreate") | Create an object of a report parameter. |
| [ReportSubscribe](imtadminapi_reportsubscribe.md "ReportSubscribe") | Subscribe to events associated with the configuration of reports. |
| [ReportUnsubscribe](imtadminapi_reportunsubscribe.md "ReportUnsubscribe") | Unsubscribe from events associated with the configuration of reports. |
| [ReportUpdate](imtadminapi_reportupdate.md "ReportUpdate") | Add or update a report configuration. |
| [ReportUpdateBatch](imtadminapi_reportupdatebatch.md "ReportUpdateBatch") | Add or edit multiple report configurations. |
| [ReportDelete](imtadminapi_reportdelete.md "ReportDelete") | Delete a report configuration by the name or index. |
| [ReportDeleteBatch](imtadminapi_reportdeletebatch.md "ReportDeleteBatch") | Delete multiple report configurations. |
| [ReportShift](imtadminapi_reportshift.md "ReportShift") | Change the position of a report configuration in the list. |
| [ReportTotal](imtadminapi_reporttotal.md "ReportTotal") | The total number of report configurations available in the platform. |
| [ReportNext](imtadminapi_reportnext.md "ReportNext") | Get a report configuration by the index. |
| [ReportGet](imtadminapi_reportget.md "ReportGet") | Get a report configuration by the name. |
| [ReportModuleTotal](imtadminapi_reportmoduletotal.md "ReportModuleTotal") | The total number of configurations of report modules available in the platform. |
| [ReportModuleNext](imtadminapi_reportmodulenext.md "ReportModuleNext") | Get a report module by the index. |
| [ReportModuleGet](imtadminapi_reportmoduleget.md "ReportModuleGet") | Get a report module configuration by the name. |

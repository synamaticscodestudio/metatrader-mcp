IMTReportSeries
The IMTReportSeries interface is used for working with data series for creating charts. It contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtreportseries_release.md "Release") | Delete the current object. |
| [Assign](imtreportseries_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtreportseries_clear.md "Clear") | Clear an object. |
| [Title](imtreportseries_title.md "Title") | Get and set a data series header. |
| [Type](imtreportseries_type.md "Type") | Get and set a data series type. |
| [Flags](imtreportseries_flags.md "Flags") | Get and set data series flags. |
| [Color](imtreportseries_color.md "Color") | Get the color used for a data series display on a chart. |
| [Tooltip](imtreportseries_tooltip.md "Tooltip") | Get and set a tooltip text for a series. |
| [ValueClear](imtreportseries_valueclear.md "ValueClear") | Clear all series values. |
| [ValueTotal](imtreportseries_valuetotal.md "ValueTotal") | Get the total amount of values in a series. |
| [ValueAdd](imtreportseries_valueadd.md "ValueAdd") | Add a value to a series. |
| [ValueAddInt](imtreportseries_valueaddint.md "ValueAddInt") | Add integer value to a series. |
| [ValueAddDouble](imtreportseries_valueadddouble.md "ValueAddDouble") | Add a value with a floating point to a series. |
| [ValueUpdate](imtreportseries_valueupdate.md "ValueUpdate") | Change the value at the indicated series position. |
| [ValueUpdateInt](imtreportseries_valueupdateint.md "ValueUpdateInt") | Change the integer value at the indicated series position. |
| [ValueUpdateDouble](imtreportseries_valueupdatedouble.md "ValueUpdateDouble") | Change the value with a floating point at the indicated series position. |
| [ValueNext](imtreportseries_valuenext.md "ValueNext") | Get the value at the indicated series position. |
| [ValueNextInt](imtreportseries_valuenextint.md "ValueNextInt") | Get the integer value at the indicated series position. |
| [ValueNextDouble](imtreportseries_valuenextdouble.md "ValueNextDouble") | Get the value with a floating point at the indicated series position. |
| [ValueShift](imtreportseries_valueshift.md "ValueShift") | Shift a value in a series. |
| [ValueDelete](imtreportseries_valuedelete.md "ValueDelete") | Delete the value at the indicated series position. |
| [ValueDescription](imtreportseries_valuedescription.md "ValueDescription") | Get and set a description for a series value. |

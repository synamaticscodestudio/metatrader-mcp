Positions
Functions allow to view the database of positions, as well subscribe and unsubscribe from events associated with changes in the database of positions.
The following functions are available for this purpose:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [PositionCreate](imtmanagerapi_positioncreate.md "PositionCreate") | Create an object of a trade position. |
| [PositionCreateArray](imtmanagerapi_positioncreatearray.md "PositionCreateArray") | Create an object of the array of trade positions. |
| [PositionSubscribe](imtmanagerapi_positionsubscribe.md "PositionSubscribe") | Subscribe to the events associated with changes in the database of positions. |
| [PositionUnsubscribe](imtmanagerapi_positionunsubscribe.md "PositionUnsubscribe") | Unsubscribe from the events associated with changes in the database of positions. |
| [PositionGet](imtmanagerapi_positionget.md "PositionGet") | Get an open trade position or an array of positions. |
| [PositionGetByGroup](imtmanagerapi_positiongetbygroup.md "PositionGetByGroup") | Get all currently opened positions for one or several client groups. |
| [PositionGetByLogins](imtmanagerapi_positiongetbylogins.md "PositionGetByLogins") | Receive open positions by the list of logins. |
| [PositionGetByTicket](imtmanagerapi_positiongetbyticket.md "PositionGetByTicket") | Get an open trade position by the ticket. |
| [PositionGetByTickets](imtmanagerapi_positiongetbytickets.md "PositionGetByTickets") | Receive open positions by the list of tickets. |
| [PositionGetBySymbol](imtmanagerapi_positiongetbysymbol.md "PositionGetBySymbol") | Receive currently open positions by groups and symbol. |
| [PositionRequest](imtmanagerapi_positionrequest.md "PositionRequest") | Request from the server open positions by login. |
| [PositionRequestByGroup](imtmanagerapi_positionrequestbygroup.md "PositionRequestByGroup") | Request from the server open positions related to a client group. |
| [PositionRequestByGroupSymbol](imtmanagerapi_positionrequestbygroupsymbol.md "PositionRequestByGroupSymbol") | Request open positions from the server by group and symbol. |
| [PositionRequestByLogins](imtmanagerapi_positionrequestbylogins.md "PositionRequestByLogins") | Request from the server open positions by the list of logins. |
| [PositionRequestByLoginsSymbol](imtmanagerapi_positionrequestbyloginsymbol.md "PositionRequestByLoginsSymbol") | Request open positions from the server by list of logins and symbol. |
| [PositionRequestByTickets](imtmanagerapi_positionrequestbytickets.md "PositionRequestByTickets") | Request from the server open positions by the list of tickets. |
| [PositionUpdate](imtmanagerapi_positionupdate.md "PositionUpdate") | Update a position. |
| [PositionUpdateBatch](imtmanagerapi_positionupdatebatch.md "PositionUpdateBatch") | Update positions in a server database in bulk. |
| [PositionUpdateBatchArray](imtmanagerapi_positionupdatebatcharray.md "PositionUpdateBatchArray") | Update positions in a server database in bulk. |
| [PositionDelete](imtmanagerapi_positiondelete.md "PositionDelete") | Delete a position. |
| [PositionDeleteByTicket](imtmanagerapi_positiondeletebyticket.md "PositionDeleteByTicket") | Delete a position by ticket. |
| [PositionDeleteBatch](imtmanagerapi_positiondeletebatch.md "PositionDeleteBatch") | Delete positions from the server database in bulk. |
| [PositionSplit](imtmanagerapi_positionsplit.md "PositionSplit") | Split trading positions. |

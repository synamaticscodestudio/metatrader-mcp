IMTConFeederTranslate
The IMTConFeederTranslate interface contains methods for managing conversion of symbols and quotes received from a data feed.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconfeedertranslate_release.md "Release") | Get the parameter of the data feed module by name. |
| [Assign](imtconfeedertranslate_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconfeedertranslate_clear.md "Clear") | Clear an object. |
| [Source](imtconfeedertranslate_source.md "Source") | Get and set the symbol name in the data feed. |
| [Symbol](imtconfeedertranslate_symbol.md "Symbol") | Get and set the symbol name in the trading platform. |
| [BidMarkup](imtconfeedertranslate_bidmarkup.md "BidMarkup") | Get and set a correction to the Bid price arriving for a symbol from a data feed. |
| [AskMarkup](imtconfeedertranslate_askmarkup.md "AskMarkup") | Get and set a correction to the Ask price arriving for a symbol from a data feed. |
| [Digits](imtconfeedertranslate_digits.md "Digits") | Get the number of digits after the decimal point in the price of the symbol that receives quotes. |

Mail Database Functions
Using the functions, you can create and send emails via the internal mail system, as well as subscribe and unsubscribe from events with changes in the mail database.
The following functions are available for working with the mail system:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [MailCreate](imtadminapi_mailcreate.md "MailCreate") | Create a message in the internal mail system. |
| [MailSubscribe](imtadminapi_mailsubscribe.md "MailSubscribe") | Subscribe to events associated with changes in the mail database. |
| [MailUnsubscribe](imtadminapi_mailunsubscribe.md "MailUnsubscribe") | Undubscribe from events associated with changes in the mail database. |
| [MailTotal](imtadminapi_mailtotal.md "MailTotal") | Get the total number of emails in the manager's mailbox. |
| [MailNext](imtadminapi_mailnext.md "MailNext") | Get an email by a position in the mailbox. |
| [MailDelete](imtadminapi_maildelete.md "MailDelete") | Delete an email by a position in the mailbox. |
| [MailDeleteId](imtadminapi_maildeleteid.md "MailDeleteId") | Delete an email by an ID. |
| [MailSend](imtadminapi_mailsend.md "MailSend") | Send emails via the internal mail system. |
| [MailBodyRequest](imtadminapi_mailbodyrequest.md "MailBodyRequest") | Get an email body. |

IMTConKYCCountry
The IMTConKYCCountry class contains methods for getting and editing country settings in the KYC provider:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconkyccountry_release.md "Release") | Delete the current object. |
| [Assign](imtconkyccountry_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconkyccountry_clear.md "Clear") | Clear an object. |
| [CountryCode](imtconkyccountry_countrycode.md "CountryCode") | Get and set the country. |

Administrator Interface IMTAdminAPI
The IMTAdminAPI interface provides access to the command of the MetaTrader 5 Administrator Terminal. Using this interface in an application, you can manage settings of a trading platform, work with databases of users, mails, prices and trade operations.
To be able to use the IMTAdminAPI interface, a manager account that is used for [connecting to the server](imtadminapi_connect.md "Connect") must have the "Connect using MetaTrader 5 Administrator" permission ([IMTConManager::RIGHT_ADMIN](imtconmanager_enum.htm#enmanagerrights)).

IMTConSubscriptionSink
The IMTConSubscriptionSink class contains event handlers related to subscription configuration changes:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnSubscriptionCfgAdd](imtconsubscriptionsink_onsubscriptioncfgadd.md "OnAutomationAdd") | Event handler for adding a new subscription configuration. |

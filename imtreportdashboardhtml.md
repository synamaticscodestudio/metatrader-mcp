IMTReportDashboardHtml
The IMTDataset interface is used for working with dashboard HTML content. It contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Assign](imtreportdashboardhtml_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtreportdashboardhtml_clear.md "Clear") | Clear an object. |
| [Write](imtreportdashboardhtml_write.md "Write") | Output a formatted sting into an HTML page. |
| [WriteString](imtreportdashboardhtml_writestring.md "WriteString") | Output an unformatted string into an HTML page (faster output). |
| [WriteSafe](imtreportdashboardhtml_writesafe.md "WriteSafe") | Output a string into HTML with HTML service symbols screening. |
| [WriteChart](imtreportdashboardhtml_writechart.md "WriteChart") | Display a chart on a dashboard. |
| [TplLoad](imtreportdashboardhtml_tplload.md "TplLoad") | Load a template from the submitted line. |
| [TplLoadFile](imtreportdashboardhtml_tplloadfile.md "TplLoadFile") | Load a template from a file. |
| [TplLoadResource](imtreportdashboardhtml_tplloadresource.md "TplLoadResource") | Load a template from a resource. |
| [TplNext](imtreportdashboardhtml_tplnext.md "TplNext") | Get the next macros (predetermined tag) from a template for processing. |
| [TplProcess](imtreportdashboardhtml_tplprocess.md "TplProcess") | Set the feature of the processing of all tags inside the current macros (predetermined tag). |

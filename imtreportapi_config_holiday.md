Holiday Configuration
To get holidays configuration, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [HolidayCreate](imtreportapi_holidaycreate.md "HolidayCreate") | Create an object of the configuration of holidays. |
| [HolidayTotal](imtreportapi_holidaytotal.md "HolidayTotal") | The total number of configurations of holidays available in the platform. |
| [HolidayNext](imtreportapi_holidaynext.md "HolidayNext") | Gets a holiday configuration with the specified index. |

Common Configuration
The Web API provides the following requests for getting common settings of a trading platform:
| Request | Description |
| --- | --- |
| --- | --- |
| [/api/common/get](webapi_common_get.md "Get") | Get the common platform configuration. |
| [/api/common/set](webapi_common_set.md "Set") | Update the common platform configuration. |

The format, in which the data about the common configuration is passed, are described in the ["Data structure"](webapi_common_data_structure.md "Data Structure") section.

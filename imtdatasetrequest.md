IMTDatasetRequest
The IMTDatasetRequest interface is a description of a [database request](reportapi_dataset.htm#request). It consists of a set of [IMTDatasetField](imtdatasetfield.md "IMTDatasetField") objects. The interface contains the following methods:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtdatasetrequest_release.md "Release") | Delete the current object. |
| [Assign](imtdatasetrequest_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtdatasetrequest_clear.md "Clear") | Clear an object. |
| [FieldCreate](imtdatasetrequest_fieldcreate.md "FieldCreate") | Create a field object. |
| [FieldCreateReference](imtdatasetrequest_fieldcreatereference.md "FieldCreateReference") | Create a reference to the object of the field at the specified position. |
| [FieldAdd](imtdatasetrequest_fieldadd.md "FieldAdd") | Add a field object to the request. |
| [FieldUpdate](imtdatasetrequest_fieldupdate.md "FieldUpdate") | Change a field object in the request. |
| [FieldDelete](imtdatasetrequest_fielddelete.md "FieldDelete") | Delete a field object from the request. |
| [FieldClear](imtdatasetrequest_fieldclear.md "FieldClear") | Clear the request description. |
| [FieldShift](imtdatasetrequest_fieldshift.md "FieldShift") | Change the position of the field description in the request. |
| [FieldTotal](imtdatasetrequest_fieldtotal.md "FieldTotal") | Get the total number of fields in a request. |
| [FieldNext](imtdatasetrequest_fieldnext.md "FieldNext") | Get a field description in a request based on its index. |

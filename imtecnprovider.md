IMTECNProvider
This interface provides access to the parameter of the [provider](https://support.metaquotes.net/en/docs/mt5/platform/administration/ecn/ecn_matching#providers) via which the requests are executed.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtecnprovider_release.md "Release") | Delete the current object. |
| [Assign](imtecnprovider_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtecnprovider_clear.md "Clear") | Clear an object. |
| [ID](imtecnprovider_id.md "ID") | Get the record ID. |
| [ProviderID](imtecnprovider_providerid.md "ProviderID") | Get the ID of the provider through which the order is forwarded to the external system. |
| [Name](imtecnprovider_name.md "Name") | Get the name of the provider through which the order is forwarded to the external system. |
| [Module](imtecnprovider_module.md "Module") | Get the name of the gateway through which the order is forwarded to the external system. |
| [Version](imtecnprovider_version.md "Version") | Get the version of the gateway through which the order is forwarded to the external system. |

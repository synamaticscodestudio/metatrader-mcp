Configuration of Plugins
To get the configurations of plugins, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [PluginCreate](imtreportapi_plugincreate.md "PluginCreate") | Create an object of the plugin configuration. |
| [PluginModuleCreate](imtreportapi_pluginmodulecreate.md "PluginModuleCreate") | Create an object of the plugin module configuration. |
| [PluginParamCreate](imtreportapi_pluginparamcreate.md "PluginParamCreate") | Create an object of the plugin parameter. |
| [PluginTotal](imtreportapi_plugintotal.md "PluginTotal") | The total number of plugin configurations available in the platform. |
| [PluginNext](imtreportapi_pluginnext.md "PluginNext") | Get the plugin configuration by the index. |
| [PluginGet](imtreportapi_pluginget.md "PluginGet") | Get the plugin configuration by the name. |
| [PluginModuleTotal](imtreportapi_pluginmoduletotal.md "PluginModuleTotal") | The total number of configurations of plugin modules (DLL files) available in the platform. |
| [PluginModuleNext](imtreportapi_pluginmodulenext.md "PluginModuleNext") | Get a plugin module by the index. |
| [PluginModuleGet](imtreportapi_pluginmoduleget.md "PluginModuleGet") | Get the plugin module configuration by the name. |

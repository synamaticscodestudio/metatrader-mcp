Custom Functions
MetaTrader 5 Server API provides the possibility of extending the protocol for running custom commands on the server. The following functions are provided for this purpose:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [CustomSubscribe](imtserverapi_customsubscribe.md "CustomSubscribe") | Subscribe to events and hooks associated with the execution of custom functions. |
| [CustomUnsubscribe](imtserverapi_customunsubscribe.md "CustomUnsubscribe") | Unsubscribe from events and hooks associated with the execution of custom functions. |
| [CustomCreateStream](imtserverapi_customcreatestream.md "CustomCreateStream") | Create an object of a byte stream. |
| [CustomCommand](imtserverapi_customcommand.md "CustomCommand") | Send a custom command to another server within the cluster. |

Tools
The MetaTrader 5 API provides a number of functions that help to facilitate a number of operations when writing applications. Additional functions are described in a number of classes included in a single header file MT5APITools.h, which is located in the /Common folder.
The additional functions are described by the following classes:
| Class | Description |
| --- | --- |
| --- | --- |
| [SMTFormat](smtformat.md "SMTFormat") | String formatting functions. |
| [SMTMath](smtmath.md "SMTMath") | The functions of mathematical transformations. |
| [SMTSearch](smtsearch.md "SMTSearch") | Sort and search functions. |
| [CMTArrayBase](cmtarraybase.md "CMTArrayBase") | A dynamic array class. |
| [CMTStr](cmtstr.md "CMTStr") | A string class. |
| [CMTSync](cmtsync.md "CMTSync") | A class of critical section of synchronization between threads. |
| [CMTThread](cmtthread.md "CMTThread") | Thread control class. |
| [SMTTime](smttime.md "SMTTime") | Time conversion class. |
| [CMTFile](cmtfile.md "CMTFile") | File operations class. |

Configuration of Managers
Functions from this section allow obtaining the description of own manager account, including access permissions for certain data and platform functions.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ManagerCreate](imtmanagerapi_managercreate.md "ManagerCreate") | Create a manager configuration object. |
| [ManagerAccessCreate](imtmanagerapi_manageraccesscreate.md "ManagerAccessCreate") | Create an object of access list configuration by IP addresses for a manager. |
| [ManagerReportCreate](imtmanagerapi_managerreportcreate.md "ManagerReportCreate") | Create an object for the manager's permission to access the report. |
| [ManagerCurrent](imtmanagerapi_managercurrent.md "ManagerCurrent") | Get the configuration of the current manager account. |

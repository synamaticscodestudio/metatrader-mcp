Configuration of Managers
Functions described in this section allow managing the configurations of managers, as well subscribe and unsubscribe from events associated with their change.
| Function | Purpose |
| --- | --- |
| --- | --- |
| [ManagerCreate](imtserverapi_managercreate.md "ManagerCreate") | Create a manager configuration object. |
| [ManagerAccessCreate](imtserverapi_manageraccesscreate.md "ManagerAccessCreate") | Create an object of configuration of an access list by IP addresses for a manager. |
| [ManagerReportCreate](imtserverapi_managerreportcreate.md "ManagerReportCreate") | Create an object for the manager's permission to access the report. |
| [ManagerSubscribe](imtserverapi_managersubscribe.md "ManagerSubscribe") | Subscribe to events and hooks associated with a manager configuration. |
| [ManagerUnsubscribe](imtserverapi_managerunsubscribe.md "ManagerUnsubscribe") | Unsubscribe from events and hooks associated with a manager configuration. |
| [ManagerAdd](imtserverapi_manageradd.md "ManagerAdd") | Add or update a manager configuration. |
| [ManagerDelete](imtserverapi_managerdelete.md "ManagerDelete") | Deletes a manager configuration by the index. |
| [ManagerShift](imtserverapi_managershift.md "ManagerShift") | Change the position of a manager configuration in the list. |
| [ManagerTotal](imtserverapi_managertotal.md "ManagerTotal") | The total number of manager configurations available in the platform. |
| [ManagerNext](imtserverapi_managernext.md "ManagerNext") | Get a manager configuration with the specified index. |
| [ManagerGet](imtserverapi_managerget.md "ManagerGet") | Get a manager configuration with the specified login. |

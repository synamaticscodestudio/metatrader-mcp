Connecting to the Server
This section contains descriptions of methods for connecting and disconnecting an application from the server. The application is connected to the platform in one of the [pumping mode](imtadminapi_enpumpmodes.md "Pumping Mode"), which indicates what information the application will continuously receive from the server.
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Connect](imtadminapi_connect.md "Connect") | Connect to the trading platform. |
| [Disconnect](imtadminapi_disconnect.md "Disconnect") | Disconnect from the trading platform. |
| [Subscribe](imtadminapi_subscribe.md "Subscribe") | Subscribe to common events of the [IMTAdminAPI](imtadminapi.md "Administrator Interface"). |
| [Unsubscribe](imtadminapi_unsubscribe.md "Unsubscribe") | Unsubscribe from common events of the [IMTAdminAPI](imtadminapi.md "Administrator Interface"). |
| [ProxySet](imtadminapi_proxyset.md "ProxySet") | Set parameters of connection to the trading platform through a proxy server. |

IMTConLeverageSink
The IMTConLeverageSink class contains methods for tracking and intercepting event related to changes in [floating margin](https://support.metaquotes.net/ru/docs/mt5/platform/administration/leverages) configurations:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [OnLeverageAdd](imtconleveragesink_onleverageadd.md "OnAutomationAdd") | Handler for the event of adding a new floating margin configuration. |
| [OnLeverageUpdate](imtconleveragesink_onleverageupdate.md "OnLeverageUpdate") | Handler for the event of updating a floating margin configuration. |
| [OnLeverageDelete](imtconleveragesink_onleveragedelete.md "OnLeverageDelete") | Handler for the event of deleting a floating margin configuration. |
| [OnLeverageSync](imtconleveragesink_onleveragesync.md "OnLeverageSync") | Handler for the event of synchronizing a floating margin configuration. |
| [HookLeverageAdd](imtconleveragesink_hookleverageadd.md "OnAutomationAdd") | Hook for adding a new floating margin configuration. |
| [HookLeverageUpdate](imtconleveragesink_hookleverageupdate.md "HookLeverageUpdate") | Hook for editing a floating margin configuration. |
| [HookLeverageDelete](imtconleveragesink_hookleveragedelete.md "HookLeverageDelete") | Hook for deleting a floating margin configuration. |

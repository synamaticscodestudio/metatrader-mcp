IMTConFundInvestor
The IMTConFundInvestor class methods for getting and changing the list of [fund investors](https://support.metaquotes.net/en/docs/mt5/platform/administration/fund_etf#investors):
| Method | Purpose |
| --- | --- |
| --- | --- |
| [Release](imtconfundinvestor_release.md "Release") | Delete the current object. |
| [Assign](imtconfundinvestor_assign.md "Assign") | Assign a passed object to the current one. |
| [Clear](imtconfundinvestor_clear.md "Clear") | Clear an object. |
| [Login](imtconfundinvestor_login.md "Login") | Get and set the login of the account used by the fund investor. |
| [Name](imtconfundinvestor_name.md "Name") | Get the name of the fund investor. |
| [SharesVolume](imtconfundinvestor_sharesvolume.md "SharesVolume") | Get the investor's share size. |

Deals
Using the functions, you can request a server database of deals and create new deals. The following functions are available for working with deals:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DealCreate](imtadminapi_dealcreate.md "DealCreate") | Create an object of a deal. |
| [DealCreateArray](imtadminapi_dealcreatearray.md "DealCreateArray") | Create an object of the array of deals. |
| [DealRequest](imtadminapi_dealrequest.md "DealRequest") | Get a deal by a ticket and login. |
| [DealRequestByGroup](imtadminapi_dealrequestbygroup.md "DealRequestByGroup") | Receive deals related to a group of accounts. |
| [DealRequestByGroupSymbol](imtadminapi_dealrequestbygroupsymbols.md "DealRequestByGroupSymbol") | Get deals by group and symbol. |
| [DealRequestByLogins](imtadminapi_dealrequestbylogins.md "DealRequestByLogins") | Receive deals by the list of logins. |
| [DealRequestByLoginsSymbol](imtadminapi_dealrequestbyloginssymbol.md "DealRequestByLoginsSymbol") | Get deals by a list of logins and symbol. |
| [DealRequestByTickets](imtadminapi_dealrequestbytickets.md "DealRequestByTickets") | Receive deals by the list of tickets. |
| [DealRequestPage](imtadminapi_dealrequestpage.md "DealRequestPage") | Get client deals with a paged output. |
| [DealAdd](imtadminapi_dealadd.md "DealAdd") | Add a deal to the server database. |
| [DealAddBatch](imtadminapi_dealaddbatch.md "DealAddBatch") | Add a batch of deals to the server database. |
| [DealAddBatchArray](imtadminapi_dealaddbatcharray.md "DealAddBatchArray") | Add a batch of deals to the server database. |
| [DealUpdate](imtadminapi_dealupdate.md "DealUpdate") | Update a deal. |
| [DealUpdateBatch](imtadminapi_dealupdatebatch.md "DealUpdateBatch") | Update deals in a server database in bulk. |
| [DealUpdateBatchArray](imtadminapi_dealupdatebatcharray.md "DealUpdateBatchArray") | Update deals in a server database in bulk. |
| [DealDelete](imtadminapi_dealdelete.md "DealDelete") | Delete a deal. |
| [DealDeleteBatch](imtadminapi_dealdeletebatch.md "DealDeleteBatch") | Delete deals from the server database in bulk. |
| [DealBackupList](imtadminapi_dealbackuplist.md "DealBackupList") | Request the dates of backup databases of deals for the specified time range. |
| [DealBackupRequest](imtadminapi_dealbackuprequest.md "DealBackupRequest") | Request deals from a backup database. |
| [DealBackupRestore](imtadminapi_dealbackuprestore.md "DealBackupRestore") | Recover a deal from a backup database. |
| [DealPerform](imtadminapi_dealperform.md "DealPerform") | Perform a deal on the client's account. This method performs a market buy or sell operation on the account as if it were performed by the client through the terminal. |
| [DealPerformBatch](imtadminapi_dealperformbatch.md "DealPerformBatch") | Perform multiple deals on the client's account. This method performs market buy or sell operations on the account as if they were performed by the client through the terminal. |
| [DealPerformBatchArray](imtadminapi_dealperformbatcharray.md "DealPerformBatchArray") | Perform multiple deals on the client's account. This method performs market buy or sell operations on the account as if they were performed by the client through the terminal. |

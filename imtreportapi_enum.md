IMTReportAPI::EnHtmlSafeFlags
HTML code generation safe mode setting flags are enumerated in IMTReportAPI::EnHtmlSafeFlags.
| ID | Value | Description |
| --- | --- | --- |
| --- | --- | --- |
| HTML_SAFE_NONE | 0x0 | No flags. |
| HTML_SAFE_USENOBSP | 0x1 | Empty lines generated by the [HtmlWriteSafe](imtreportapi_htmlwritesafe.md "HtmlWriteSafe") method are replaced with "&nbsp;" nonbreaking space symbol when the flag is turned on. |

| TPL_PROCESS_NONE | 0x0 | No flags. |

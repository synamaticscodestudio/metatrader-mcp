Gateway Configuration
To get the configuration of gateways, the following functions are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [GatewayCreate](imtreportapi_gatewaycreate.md "GatewayCreate") | Create an object of the gateway configuration. |
| [GatewayModuleCreate](imtreportapi_gatewaymodulecreate.md "GatewayModuleCreate") | Create an object of configuration of the gateway module. |
| [GatewayParamCreate](imtreportapi_gatewayparamcreate.md "GatewayParamCreate") | Create an object of the gateway parameter. |
| [GatewayTranslateCreate](imtreportapi_gatewaytranslatecreate.md "GatewayTranslateCreate") | Create an object of the parameter for converting the information received by the gateway. |
| [GatewayTotal](imtreportapi_gatewaytotal.md "GatewayTotal") | The total number of gateway configurations available in the platform. |
| [GatewayNext](imtreportapi_gatewaynext.md "GatewayNext") | Get the gateway configuration by the index. |
| [GatewayGet](imtreportapi_gatewayget.md "GatewayGet") | Get the gateway configuration by the name. |
| [GatewayModuleTotal](imtreportapi_gatewaymoduletotal.md "GatewayModuleTotal") | The total number of configurations of gateway modules available in the platform. |
| [GatewayModuleNext](imtreportapi_gatewaymodulenext.md "GatewayModuleNext") | Get the gateway module by the index. |
| [GatewayModuleGet](imtreportapi_gatewaymoduleget.md "GatewayModuleGet") | Get the gateway module configuration by the name. |

Configuration of Spreads
Functions allow managing spreads, as well subscribe and unsubscribe from events associated with their change.
The following functions for managing spreads are available:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [SpreadCreate](imtgatewayapi_spreadcreate.md "SpreadCreate") | Create an object of the configuration of a spread. |
| [SpreadLegCreate](imtgatewayapi_spreadlegcreate.md "SpreadLegCreate") | Create an object of the configuration of a spread leg. |
| [SpreadSubscribe](imtgatewayapi_spreadsubscribe.md "SpreadSubscribe") | Subscribe to events and hooks associated with the configuration of spreads. |
| [SpreadUnsubscribe](imtgatewayapi_spreadunsubscribe.md "SpreadUnsubscribe") | Unsubscribe from events and hooks associated with spread configuration. |
| [SpreadAdd](imtgatewayapi_spreadadd.md "SpreadAdd") | Add or update a spread configuration. |
| [SpreadDelete](imtgatewayapi_spreaddelete.md "SpreadDelete") | Deleting a spread configuration by the index. |
| [SpreadShift](imtgatewayapi_spreadshift.md "SpreadShift") | Change the position of a spread configuration in the list. |
| [SpreadTotal](imtgatewayapi_spreadtotal.md "SpreadTotal") | The total number of spread configurations available in the platform. |
| [SpreadNext](imtgatewayapi_spreadnext.md "SpreadNext") | Receiving a spread configuration by the index. |
| [SpreadGet](imtgatewayapi_spreadget.md "SpreadGet") | Receiving a spread configuration by the identifier. |

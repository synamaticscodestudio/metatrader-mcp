Groups
Use the methods described in this section to obtain the parameters of client groups on the server:
| Method | Purpose |
| --- | --- |
| --- | --- |
| [GroupAdd](net_groupadd.md "GroupAdd") | Add or change a group configuration on the server. |
| [GroupDelete](net_groupdelete.md "GroupDelete") | Delete a group configuration with the specified name. |
| [GroupTotal](net_grouptotal.md "GroupTotal") | Get the number of groups created on the trade server. |
| [GroupNext](net_groupnext.md "GroupNext") | Get the group configuration by its index in the list of groups of the platform. |
| [GroupGet](net_groupget.md "GroupGet") | Get the group configuration by its name. |

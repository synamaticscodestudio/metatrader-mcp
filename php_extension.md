Protocol Extension
MetaTrader 5 Web API package includes the example of [custom commands](webapi_protocol_extension.md "Protocol Extension") implementation.
Sending commands and getting results of their execution from a server must be implemented to extend the protocol (create custom commands). Besides, handling of such commands must be implemented on a trading server as a plugin. Source code of such plugin is included in the MetaTrader 5 Server API package.
[A few simple actions](php_extension_install.md "Setup") must be performed to launch the example.

Deals
The function allow observing a database of deals on a trade server. The following functions are available for working with deals:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [DealCreate](imtreportapi_dealcreate.md "DealCreate") | Create an object of a deal. |
| [DealCreateArray](imtreportapi_dealcreatearray.md "DealCreateArray") | Create an object of the array of deals. |
| [DealGet](imtreportapi_dealget.md "DealGet") | Get clients deals. |
| [DealSelect](imtreportapi_dealselect.md "DealSelect") | Request deals from a database according to specified criteria. |

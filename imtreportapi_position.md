Positions
The functions allow to observe a database of positions on a server. The following functions are available for this purpose:
| Function | Purpose |
| --- | --- |
| --- | --- |
| [PositionCreate](imtreportapi_positioncreate.md "PositionCreate") | Create an object of a trade position. |
| [PositionCreateArray](imtreportapi_positioncreatearray.md "PositionCreateArray") | Create an object of the array of trade positions. |
| [PositionGet](imtreportapi_positionget.md "PositionGet") | Get clients positions. |
| [PositionGetByTicket](imtreportapi_positiongetbyticket.md "PositionGetByTicket") | Get a trade position by the ticket. |
| [PositionSelect](imtreportapi_positionselect.md "PositionSelect") | Request trading positions from a database according to specified criteria. |
